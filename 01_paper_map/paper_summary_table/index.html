五篇論文速覽 Paper Summary Table

來源資料夾共 5 個 PDF。這頁用一張表把它們一眼看完，再用教學角度整理它們的分工： 哪篇是 ISF 核心、哪篇延伸到 ring、哪篇講 injection、哪篇其實離題，以及哪些公式 其實是同一件事的不同寫法、哪些符號定義不同需要統一。

誠實聲明（先講）：這 5 篇裡，4 篇是 Hajimiri 系列的 oscillator phase noise／ injection 論文，1 篇（Hajimiri_ISCS_98.pdf = [P5]）其實是 cross-coupled sense amplifier 論文、與 ISF 無關，本站只當邊角註解。另外，常被一起講的 PPV / adjoint / PSS / PNoise / Floquet 其實沒有任何一篇專屬論文在這 5 個 PDF 內——它們屬外部文獻，誠實標註於下。

教學角度：誰負責教什麼​

哪篇是 ISF 核心基礎​

[P1] 是整站的地基。LTV/ISF 的所有概念——Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ)、qmaxq_{max}qmax​ normalization、卷積式相位響應、傅立葉降頻、1/f21/f^21/f2/1/f31/f^31/f3 封閉式——全部出自這裡。 02 章（foundations）與 03 章（isf core theory）幾乎逐節對應 [P1] 第 III–IV 節， lab_01/02/04/05/06/07 都在重現 [P1] 的圖與公式。

哪篇延伸到 ring​

[P2] 把 [P1] 的通用理論落到 ring oscillator：同一組 Γrms2/qmax2\Gamma_{rms}^2/q_{max}^2Γrms2​/qmax2​ 邏輯給出累積 jitter（claim C6）、Γrms∝N−3/4\Gamma_{rms}\propto N^{-3/4}Γrms​∝N−3/4（claim C8）， 與「固定功率/頻率下 ring phase noise 幾乎與 NNN 無關」（claim C7）。06 章 lc_vs_ring、symmetry 與 lab_03 用它。

哪篇關於 adjoint / PPV / PSS / PNoise / simulation​

老實說：這 5 篇裡沒有任何一篇是專門講 PPV/adjoint/PSS/PNoise 的。

  • PPV（perturbation projection vector，擾動投影向量）／adjoint method／Floquet theory 是 ISF 背後嚴謹的數學基礎，但它們來自更廣的文獻（如 Demir–Mehrotra–Roychowdhury 2000、Kaertner），不在下載的 5 篇 PDF 內。本站把它們 當標準外部文獻補充，明確標註於 effective_isf （對應 claim C13）。
  • PSS（periodic steady-state）／PNoise 是商用模擬器（如 SpectreRF）萃取 ISF／ phase noise 的數值方法，同樣不是這 5 篇的任何一篇的主題；本站只在概念上引用， 不宣稱出自下載 PDF。
  • 與「simulation」最相關的，反而是 [P1] 自己提到「實際電路要靠 transient impulse response 或 adjoint/PSS 萃取 ISF」這個limitation——也就是說，模擬萃取被提及， 但沒有專屬論文。本站的所有 lab 都是教學用 toy model（非 transistor-level）， 用 Python 重現概念，不是真實 PSS/PNoise 流程。

哪篇修正／批判／補充前人​

[P1] 取代 Leeson 的經驗式。 Leeson 1966 的 L\mathcal{L}L 表達式（含 2FkT/Ps2FkT/P_s2FkT/Ps​、 (ω0/2QΔω)2(\omega_0/2Q\Delta\omega)^2(ω0​/2QΔω)2、1+ω1/f3/∣Δω∣1+\omega_{1/f^3}/|\Delta\omega|1+ω1/f3​/∣Δω∣）是經驗擬合：它能畫出 1/f31/f^31/f3、1/f21/f^21/f2、floor 三段，但 FFF（noise factor）與 1/f31/f^31/f3 corner 都是事後填的 fitting 參數，沒有從第一原理算出。[P1] 的貢獻正是：用 ISF 從第一原理算出這三段， 並指出 1/f31/f^31/f3 corner 不是 device 的 1/f1/f1/f corner，而是 ω1/f(c0/c1)2\omega_{1/f}(c_0/c_1)^2ω1/f​(c0​/c1​)2 （[P1] Eq.(24)，claim C5）——這是對 Leeson 經驗模型的關鍵修正。Leeson 式本身 不是下載的 5 篇 PDF 之一，本站只當對照（見 equation_index 第 19 列，標 reference ⚠️）。

此外 [P3] 補充/推廣 Adler 1946 的 injection locking 方程；[P4] 補上 [P1]/[P3] 缺的振幅維度（APF）。Adler 與 Leeson 的原始論文都不在這 5 個 PDF 內。

哪些符號定義不同、需要統一​

不同論文對同一量用不同記號，本站一律以 notation 為準：

  • offset 頻率：[P1] 多用 Δω\Delta\omegaΔω（rad/s），datasheet/SerDes 用 Δf\Delta fΔf（Hz）； 本站兩者並用，Δω=2πΔf\Delta\omega=2\pi\Delta fΔω=2πΔf。
  • ISF 的 DC：注意 c0c_0c0​ 是傅立葉係數，ISF 的 DC值是 c0/2c_0/2c0​/2（Eq.(12)）—— 算 1/f31/f^31/f3 corner（Eq.(24)）時極易出錯，本站反覆提醒。
  • 振幅敏感度：[P4] 的 APF 寫 Λ(ϕ)\Lambda(\phi)Λ(ϕ)、單位 A−1\mathrm{A^{-1}}A−1，與 ISF 的無因次 Γ\GammaΓ 量綱不同，不可混用。
  • PPV/adjoint/Floquet：本站不在主幹用其專屬記號（如 Demir 的 v1T(t)v_1^T(t)v1T​(t)）， 只在 effective_isf 以外部文獻形式提及。

完整符號對照（含「各論文寫法／備註」一欄）見 notation 的「各論文符號對照」段。

重點回顧​

  • [P1] = ISF 核心；[P2] = ring 延伸；[P3]/[P4] = injection locking / APF 進階； [P5] = sense amplifier，與 ISF 無關（誠實標註）。
  • 沒有任何一篇是專屬 PPV/adjoint/PSS/PNoise 論文；那些屬外部文獻（如 Demir 2000）。
  • [P1] 從第一原理取代 Leeson 經驗式，並修正「1/f31/f^31/f3 corner ≠ device 1/f1/f1/f corner」。
  • 不同論文的 Γ\GammaΓ、相位方程、Γrms2/qmax2\Gamma_{rms}^2/q_{max}^2Γrms2​/qmax2​ 常是同一件事的不同視角； 符號一律以 notation 統一。
