基礎章習題（含完整解答）

先備：psd_phase_noise_jitter · lti_vs_ltv ｜ 接下來：03 核心理論章習題

這頁是 02 基礎章 的成套習題。題型涵蓋推導題（要你把式子親手導出來）、數值題 （代數字、帶單位、做 dimension check）、與設計反推題（給目標規格、反算所需參數）。

怎麼用這頁：先把題目自己做一遍，再展開「解答」對照。每題解答都遵守同一套格式： 逐步代入（帶單位）→ 結果 → dimension check → 一行 Python 驗證。Python 驗證一律 引用本站真實函式庫 simulations/common/（不杜撰 API），你可以直接貼進 REPL 跑。

涉及的核心公式（都來自規範與本章頁面，沿用同一 notation）：

  • phase→time：Δt=Δϕ2πf0\Delta t=\dfrac{\Delta\phi}{2\pi f_0}Δt=2πf0​Δϕ​（規範公式 17）
  • phase variance：σϕ2=∫f1f2Sϕ(f) df\sigma_\phi^2=\displaystyle\int_{f_1}^{f_2}S_\phi(f)\,dfσϕ2​=∫f1​f2​​Sϕ​(f)df（規範公式 18）
  • rms jitter：σt=12πf0∫f1f2Sϕ(f) df\sigma_t=\dfrac{1}{2\pi f_0}\sqrt{\displaystyle\int_{f_1}^{f_2}S_\phi(f)\,df}σt​=2πf0​1​∫f1​f2​​Sϕ​(f)df​（規範公式 19）
  • SSB↔phase PSD（小角）：L(Δf)≈12Sϕ(Δf)\mathcal{L}(\Delta f)\approx\tfrac12 S_\phi(\Delta f)L(Δf)≈21​Sϕ​(Δf)（規範公式 16）
  • Parseval：∑n=0∞cn2=1π∫02π∣Γ(x)∣2dx=2 Γrms2\displaystyle\sum_{n=0}^{\infty}c_n^2=\frac{1}{\pi}\int_0^{2\pi}\lvert\Gamma(x)\rvert^2dx=2\,\Gamma_{rms}^2n=0∑∞​cn2​=π1​∫02π​∣Γ(x)∣2dx=2Γrms2​（[P1] Eq.(20), p.185）
  • Lorentzian 線寬（FWHM）：Δf3dB=Dπ\Delta f_{3\mathrm{dB}}=\dfrac{D}{\pi}Δf3dB​=πD​，相位擴散 Var⁡[Δϕ(t)]=2D∣t∣\operatorname{Var}[\Delta\phi(t)]=2D\lvert t\rvertVar[Δϕ(t)]=2D∣t∣（規範 11.2；連 [E2] Demir 2000，不在 5 篇 PDF 內）
  • Allan 斜率對照：white FM →τ−1/2\to\tau^{-1/2}→τ−1/2、flicker FM →τ0\to\tau^{0}→τ0、random-walk FM →τ+1/2\to\tau^{+1/2}→τ+1/2（規範 11.2；Allan 為外部文獻）

題目​

習題 1（數值題）— phase ↔ time 換算​

一顆 f0=5f_0=5f0​=5 GHz 的振盪器，某瞬間 excess phase 偏移 Δϕ=5×10−4\Delta\phi=5\times10^{-4}Δϕ=5×10−4 rad。 求對應的 timing error Δt\Delta tΔt（fs），並把 Δϕ\Delta\phiΔϕ 換成「度」。

習題 2（數值題）— rms jitter 由相位變異反推​

已知某時鐘的 rms phase σϕ=14.07\sigma_\phi=14.07σϕ​=14.07 mrad（在 1→100 MHz 積分頻段內），f0=5f_0=5f0​=5 GHz。 求 rms timing jitter σt\sigma_tσt​（fs）。

習題 3（推導題 + 數值）— Parseval：由 ISF 算 Γrms\Gamma_{rms}Γrms​ 與 ∑cn2\sum c_n^2∑cn2​​

理想 LC 振盪器的 ISF 是 Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ。

(a) 用定義 Γrms2=12π∫02πΓ2(θ) dθ\Gamma_{rms}^2=\dfrac{1}{2\pi}\displaystyle\int_0^{2\pi}\Gamma^2(\theta)\,d\thetaΓrms2​=2π1​∫02π​Γ2(θ)dθ 親手算 Γrms\Gamma_{rms}Γrms​。 (b) 用 Parseval（規範公式 11）求 ∑n=0∞cn2\sum_{n=0}^{\infty}c_n^2∑n=0∞​cn2​，並說明只有哪個 cnc_ncn​ 非零。

習題 4（數值題）— PSD 積分得相位變異​

某振盪器在 offset fff 的單邊 phase PSD 在 1/f21/f^21/f2 區可寫成 Sϕ(f)=Kf2S_\phi(f)=\dfrac{K}{f^2}Sϕ​(f)=f2K​， 其中 K=10−4 rad2⋅HzK=10^{-4}\ \text{rad}^2\cdot\text{Hz}K=10−4 rad2⋅Hz（即 Sϕ(1 Hz)=10−4S_\phi(1\,\text{Hz})=10^{-4}Sϕ​(1Hz)=10−4）。求積分頻段 f1=103f_1=10^3f1​=103 Hz 到 f2=106f_2=10^6f2​=106 Hz 的相位變異 σϕ2\sigma_\phi^2σϕ2​ 與 σϕ\sigma_\phiσϕ​（mrad）。

習題 5（概念 + 推導題）— LTI vs LTV：同 impulse、不同相位​

理想 LC 的 ISF Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ。同一顆 Δq\Delta qΔq 的電荷脈衝，分別在 (a) 波形過零點（zero-crossing，θ=π/2\theta=\pi/2θ=π/2，此處 cos⁡\coscos 波形斜率最大、−sin⁡-\sin−sin 取極值） 與 (b) 波形峰值（θ=0\theta=0θ=0）注入。用 Δϕ=Γ(θ) Δq/qmax\Delta\phi=\Gamma(\theta)\,\Delta q/q_{max}Δϕ=Γ(θ)Δq/qmax​ 說明兩者相位效果差多少，並一句話講清楚這「為什麼是 LTV 而不是 LTI」。 取 Δq=1\Delta q=1Δq=1 fC、qmax=1q_{max}=1qmax​=1 pC。

習題 6（設計反推題）— Lorentzian 線寬反推 phase diffusion​

某自由運轉振盪器量到的載波 3-dB 線寬（FWHM）Δf3dB=1\Delta f_{3\mathrm{dB}}=1Δf3dB​=1 kHz。

(a) 反推 phase diffusion 係數 DDD（rad²/s）。 (b) 估計相位累積到變異 Var⁡[Δϕ]=1 rad2\operatorname{Var}[\Delta\phi]=1\ \text{rad}^2Var[Δϕ]=1 rad2（相位「跑掉約 1 rad」、相干性大致瓦解）需要多久 ttt。

習題 7（概念 + 斜率判讀題）— Allan deviation 斜率​

在 log–log 的 Allan deviation 圖 σy(τ)\sigma_y(\tau)σy​(τ) 上量到三段不同斜率： −1/2-1/2−1/2、000、+1/2+1/2+1/2。分別對應哪一種 FM 雜訊型態？並說明為何「flicker FM」會在 ADEV 上 形成一段平台（floor）。

習題 8（數值題）— 由 L(Δf)\mathcal{L}(\Delta f)L(Δf) 換 SϕS_\phiSϕ​ 再換單音 jitter​

某 spur-free 振盪器在 Δf=1\Delta f=1Δf=1 MHz 量到 L(1 MHz)=−120\mathcal{L}(1\,\text{MHz})=-120L(1MHz)=−120 dBc/Hz。 (a) 求該 offset 的 SϕS_\phiSϕ​（rad²/Hz）。 (b) 若在 1→10 MHz 這 1 decade 內 PSD 為 1/f21/f^21/f2（即 Sϕ=K/f2S_\phi=K/f^2Sϕ​=K/f2，用 (a) 定 KKK）， f0=5f_0=5f0​=5 GHz，求此頻段 rms jitter σt\sigma_tσt​（fs）。

習題 1 解答（phase ↔ time 換算）

逐步代入（帶單位）。 用 phase→time（規範公式 17）：

Δt=Δϕ2πf0=5×10−4 rad2π×5×109 Hz=5×10−43.1416×1010 s.\Delta t=\frac{\Delta\phi}{2\pi f_0}=\frac{5\times10^{-4}\ \text{rad}}{2\pi\times5\times10^{9}\ \text{Hz}}=\frac{5\times10^{-4}}{3.1416\times10^{10}}\ \text{s}.Δt=2πf0​Δϕ​=2π×5×109 Hz5×10−4 rad​=3.1416×10105×10−4​ s.Δt≈1.59×10−14 s=15.9 fs.\Delta t\approx1.59\times10^{-14}\ \text{s}=15.9\ \text{fs}.Δt≈1.59×10−14 s=15.9 fs.

換成度：

Δϕ=5×10−4×180π≈0.0286∘.\Delta\phi=5\times10^{-4}\times\frac{180}{\pi}\approx0.0286^\circ.Δϕ=5×10−4×π180​≈0.0286∘.

結果：Δt≈15.9\Delta t\approx15.9Δt≈15.9 fs，Δϕ≈0.0286∘\Delta\phi\approx0.0286^\circΔϕ≈0.0286∘。

Dimension check：[rad][rad/s]=[s]\dfrac{[\text{rad}]}{[\text{rad/s}]}=[\text{s}][rad/s][rad]​=[s] ✓（2πf02\pi f_02πf0​ 的單位是 rad/s，rad 是 無因次，相除得秒）。這正是規範第 8 節 canonical 例 A 的 timing error。

習題 3 解答（Parseval：由 ISF 算 Γrms\Gamma_{rms}Γrms​ 與 ∑cn2\sum c_n^2∑cn2​）

(a) 直接積分算 Γrms\Gamma_{rms}Γrms​。 用 sin⁡2θ\sin^2\thetasin2θ 在一個週期上平均為 12\tfrac1221​：

Γrms2=12π∫02π(−sin⁡θ)2 dθ=12π∫02πsin⁡2θ dθ=12π⋅π=12.\Gamma_{rms}^2=\frac{1}{2\pi}\int_0^{2\pi}(-\sin\theta)^2\,d\theta=\frac{1}{2\pi}\int_0^{2\pi}\sin^2\theta\,d\theta=\frac{1}{2\pi}\cdot\pi=\frac12.Γrms2​=2π1​∫02π​(−sinθ)2dθ=2π1​∫02π​sin2θdθ=2π1​⋅π=21​.Γrms=12≈0.707.\Gamma_{rms}=\frac{1}{\sqrt2}\approx0.707.Γrms​=2​1​≈0.707.

(b) 用 Parseval 求 ∑cn2\sum c_n^2∑cn2​。 規範公式 11（[P1] Eq.(20), p.185）：

∑n=0∞cn2=2 Γrms2=2×12=1.\sum_{n=0}^{\infty}c_n^2=2\,\Gamma_{rms}^2=2\times\frac12=1.n=0∑∞​cn2​=2Γrms2​=2×21​=1.

哪個 cnc_ncn​ 非零：Γ(θ)=−sin⁡θ=cos⁡(θ+π2)\Gamma(\theta)=-\sin\theta=\cos(\theta+\tfrac{\pi}{2})Γ(θ)=−sinθ=cos(θ+2π​)，是純一次諧波， 所以只有 c1=1c_1=1c1​=1（且 θ1=π/2\theta_1=\pi/2θ1​=π/2），其餘 c0=c2=⋯=0c_0=c_2=\dots=0c0​=c2​=⋯=0。驗證：∑cn2=c12=1\sum c_n^2=c_1^2=1∑cn2​=c12​=1 ✓。

結果：Γrms=1/2≈0.707\Gamma_{rms}=1/\sqrt2\approx0.707Γrms​=1/2​≈0.707、∑cn2=1\sum c_n^2=1∑cn2​=1、唯一非零係數 c1=1c_1=1c1​=1。

習題 4 解答（PSD 積分得相位變異）

逐步代入（帶單位）。 phase variance（規範公式 18），代 Sϕ(f)=K/f2S_\phi(f)=K/f^2Sϕ​(f)=K/f2：

σϕ2=∫f1f2Kf2 df=K[−1f]f1f2=K(1f1−1f2).\sigma_\phi^2=\int_{f_1}^{f_2}\frac{K}{f^2}\,df=K\left[-\frac1f\right]_{f_1}^{f_2}=K\left(\frac{1}{f_1}-\frac{1}{f_2}\right).σϕ2​=∫f1​f2​​f2K​df=K[−f1​]f1​f2​​=K(f1​1​−f2​1​).

代 K=10−4 rad2⋅HzK=10^{-4}\ \text{rad}^2\cdot\text{Hz}K=10−4 rad2⋅Hz、f1=103f_1=10^3f1​=103 Hz、f2=106f_2=10^6f2​=106 Hz：

σϕ2=10−4(1103−1106)=10−4×(10−3−10−6)=10−4×9.99×10−4=9.99×10−8 rad2.\sigma_\phi^2=10^{-4}\left(\frac{1}{10^3}-\frac{1}{10^6}\right)=10^{-4}\times(10^{-3}-10^{-6})=10^{-4}\times9.99\times10^{-4}=9.99\times10^{-8}\ \text{rad}^2.σϕ2​=10−4(1031​−1061​)=10−4×(10−3−10−6)=10−4×9.99×10−4=9.99×10−8 rad2.σϕ=9.99×10−8≈3.16×10−4 rad=0.316 mrad.\sigma_\phi=\sqrt{9.99\times10^{-8}}\approx3.16\times10^{-4}\ \text{rad}=0.316\ \text{mrad}.σϕ​=9.99×10−8​≈3.16×10−4 rad=0.316 mrad.

結果：σϕ2≈9.99×10−8 rad2\sigma_\phi^2\approx9.99\times10^{-8}\ \text{rad}^2σϕ2​≈9.99×10−8 rad2，σϕ≈0.316\sigma_\phi\approx0.316σϕ​≈0.316 mrad。

手感：1/f21/f^21/f2 積分由低頻端 f1f_1f1​ 主導（1/f1≫1/f21/f_1\gg1/f_21/f1​≫1/f2​）——這就是為什麼近載波相位 雜訊對總 jitter 貢獻最大。

Dimension check：[K]⋅[1/f]=(rad2⋅Hz)⋅(1/Hz)=rad2[K]\cdot[1/f]=(\text{rad}^2\cdot\text{Hz})\cdot(1/\text{Hz})=\text{rad}^2[K]⋅[1/f]=(rad2⋅Hz)⋅(1/Hz)=rad2 ✓ （σϕ2\sigma_\phi^2σϕ2​ 是 rad²）。

var = K*(1/f1 - 1/f2)
print(var, "rad^2 ->", np.sqrt(var)*1e3, "mrad")   # -> 9.99e-8 rad^2 -> 0.316 mrad
習題 5 解答（LTI vs LTV：同 impulse、不同相位）

逐步代入（帶單位）。 用操作型 ISF（規範公式 5）Δϕ=Γ(θ) Δq/qmax\Delta\phi=\Gamma(\theta)\,\Delta q/q_{max}Δϕ=Γ(θ)Δq/qmax​。 Δq/qmax=10−15/10−12=10−3\Delta q/q_{max}=10^{-15}/10^{-12}=10^{-3}Δq/qmax​=10−15/10−12=10−3。

(a) 過零點 θ=π/2\theta=\pi/2θ=π/2（Γ=−sin⁡(π/2)=−1\Gamma=-\sin(\pi/2)=-1Γ=−sin(π/2)=−1，敏感度最大）：

Δϕa=(−1)×10−3=−1×10−3 rad=−1 mrad.\Delta\phi_a=(-1)\times10^{-3}=-1\times10^{-3}\ \text{rad}=-1\ \text{mrad}.Δϕa​=(−1)×10−3=−1×10−3 rad=−1 mrad.

(b) 峰值 θ=0\theta=0θ=0（Γ=−sin⁡0=0\Gamma=-\sin 0=0Γ=−sin0=0，敏感度為零）：

Δϕb=0×10−3=0 rad.\Delta\phi_b=0\times10^{-3}=0\ \text{rad}.Δϕb​=0×10−3=0 rad.

結果：在過零點注入造成 ∣Δϕ∣=1\lvert\Delta\phi\rvert=1∣Δϕ∣=1 mrad 的相位跳變；在波峰注入造成 0 相位跳變 （該處電荷只改振幅、隨後被 amplitude restoring 拉回）。

為什麼是 LTV 不是 LTI：一個 LTI（線性非時變）系統的 impulse response 只跟「經過多久」 t−τt-\taut−τ 有關，跟「何時注入」τ\tauτ 無關。但這裡同樣大小的 Δq\Delta qΔq，在 θ=π/2\theta=\pi/2θ=π/2 與 θ=0\theta=0θ=0 得到完全不同的 Δϕ\Delta\phiΔϕ——響應顯式依賴絕對注入相位 Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ)，這正是 **LTV（線性時變）**的定義特徵（見 lti_vs_ltv）。

Dimension check：Γ\GammaΓ 無因次 × Δq/qmax\times\ \Delta q/q_{max}× Δq/qmax​（C/C 無因次）=== rad ✓。

from simulations.common.isf_utils import gamma_lc_ideal, impulse_to_phase_step
import numpy as np
dq, qmax = 1e-15, 1e-12
for name, theta in [("zero-crossing", np.pi/2), ("peak", 0.0)]:
    g = gamma_lc_ideal(theta)                       # -sin(theta)
    dphi = impulse_to_phase_step(dq, g, qmax)
    print(name, dphi*1e3, "mrad")                   # -> -1.0 mrad ; 0.0 mrad
習題 6 解答（Lorentzian 線寬反推 phase diffusion）

註：Lorentzian 線寬與 phase diffusion 屬外部文獻（[E2] Demir 2000），不在 5 篇 PDF 內； 公式逐字取自規範 11.2。

(a) 反推 DDD。 FWHM 線寬（規範 11.2）：

Δf3dB=Dπ⟹D=π Δf3dB=π×103 Hz=3.142×103 rad2/s.\Delta f_{3\mathrm{dB}}=\frac{D}{\pi}\quad\Longrightarrow\quad D=\pi\,\Delta f_{3\mathrm{dB}}=\pi\times10^{3}\ \text{Hz}=3.142\times10^{3}\ \text{rad}^2/\text{s}.Δf3dB​=πD​⟹D=πΔf3dB​=π×103 Hz=3.142×103 rad2/s.

(b) 估累積到 1 rad² 的時間。 相位擴散（規範 11.2）Var⁡[Δϕ(t)]=2D∣t∣\operatorname{Var}[\Delta\phi(t)]=2D\lvert t\rvertVar[Δϕ(t)]=2D∣t∣，令其 =1 rad2=1\ \text{rad}^2=1 rad2：

t=12D=12×3.142×103≈1.59×10−4 s=159 μs.t=\frac{1}{2D}=\frac{1}{2\times3.142\times10^{3}}\approx1.59\times10^{-4}\ \text{s}=159\ \mu\text{s}.t=2D1​=2×3.142×1031​≈1.59×10−4 s=159 μs.

結果：D≈3.14×103 rad2/sD\approx3.14\times10^{3}\ \text{rad}^2/\text{s}D≈3.14×103 rad2/s；相位累積到 ≈1 rad2\approx1\ \text{rad}^2≈1 rad2 約需 159 µs （即相干時間 τc∼1/(2D)\tau_c\sim1/(2D)τc​∼1/(2D) 量級）。注意 1/Δf3dB=11/\Delta f_{3\mathrm{dB}}=11/Δf3dB​=1 ms 與此同數量級—— 線寬越窄，相干時間越長，兩者倒數關係。

import numpy as np
df_3db = 1e3                       # Hz (FWHM)

print(D, "rad^2/s ;", t*1e6, "us")   # -> 3141.6 rad^2/s ; 159.2 us

習題 7 解答（Allan deviation 斜率判讀）

註：Allan variance 屬外部文獻（Allan 1966），不在 5 篇 PDF 內；斜率對照表逐字取自規範 11.2。

斜率對照（規範 11.2）。 在 log–log 的 σy(τ)\sigma_y(\tau)σy​(τ) 圖上：

量到的斜率 σy(τ)∝τ?\sigma_y(\tau)\propto\tau^{?}σy​(τ)∝τ?對應雜訊型態物理
τ−1/2\tau^{-1/2}τ−1/2white FM（white frequency modulation）頻率是白噪 → 相位是隨機漫步；平均越久越穩
τ0\tau^{0}τ0（平台/floor）flicker FM（1/f frequency）σy\sigma_yσy​ 與閘門時間 τ\tauτ 無關 → ADEV 形成平地板
τ+1/2\tau^{+1/2}τ+1/2random-walk FM（頻率隨機漫步）頻率本身漂移 → 平均越久越差，斜率上揚

為什麼 flicker FM 形成平台：把 Sy(f)∝1/fS_y(f)\propto1/fSy​(f)∝1/f（flicker FM）代入規範 11.2 的 σy2(τ)=2∫0∞Sy(f)sin⁡4(πfτ)(πfτ)2 df\sigma_y^2(\tau)=2\displaystyle\int_0^\infty S_y(f)\dfrac{\sin^4(\pi f\tau)}{(\pi f\tau)^2}\,dfσy2​(τ)=2∫0∞​Sy​(f)(πfτ)2sin4(πfτ)​df，做變數代換 u=πfτu=\pi f\tauu=πfτ，τ\tauτ 會從被積函數裡完全約掉——積分變成一個與 τ\tauτ 無關的純數常數， 所以 σy2(τ)=\sigma_y^2(\tau)=σy2​(τ)= 常數 ⇒σy(τ)∝τ0\Rightarrow\sigma_y(\tau)\propto\tau^0⇒σy​(τ)∝τ0，在 log–log 上是一條水平線（floor）。 這個 floor 是石英/原子鐘等長期穩定度的招牌特徵，標定「再怎麼延長平均時間都壓不下去」的底限。

結果：τ−1/2→\tau^{-1/2}\toτ−1/2→ white FM、τ0→\tau^{0}\toτ0→ flicker FM（floor）、τ+1/2→\tau^{+1/2}\toτ+1/2→ random-walk FM。

Dimension check：σy(τ)\sigma_y(\tau)σy​(τ) 是無因次的分數頻率穩定度（y=Δf/f0y=\Delta f/f_0y=Δf/f0​），與 τ\tauτ 的冪次關係只比斜率、本身無量綱 ✓。

import numpy as np
# 用斜率定義判讀：對某段 sigma_y(tau) 取 log-log 斜率
tau = np.logspace(-3, 1, 200)
for label, slope in [("white FM", -0.5), ("flicker FM", 0.0), ("random-walk FM", 0.5)]:
    sy = tau**slope
    fit = np.polyfit(np.log10(tau), np.log10(sy), 1)[0]
    print(label, "slope =", round(fit, 2))   # -> -0.5 ; 0.0 ; 0.5

習題 8 解答（由 L\mathcal{L}L 換 SϕS_\phiSϕ​ 再換單音 jitter）

(a) L→Sϕ\mathcal{L}\to S_\phiL→Sϕ​。 先把 dBc/Hz 還原成 linear：

Llin(1 MHz)=10−120/10=10−12 (per Hz).\mathcal{L}_{\text{lin}}(1\,\text{MHz})=10^{-120/10}=10^{-12}\ \text{(per Hz)}.Llin​(1MHz)=10−120/10=10−12 (per Hz).

Sϕ(1 MHz)=2×10−12=2×10−12 rad2/Hz.S_\phi(1\,\text{MHz})=2\times10^{-12}=2\times10^{-12}\ \text{rad}^2/\text{Hz}.Sϕ​(1MHz)=2×10−12=2×10−12 rad2/Hz.

(b) 定 KKK 再積分得 σt\sigma_tσt​。 1/f21/f^21/f2 模型 Sϕ(f)=K/f2S_\phi(f)=K/f^2Sϕ​(f)=K/f2，用 (a) 在 f=106f=10^6f=106 Hz 定 KKK：

K=Sϕ(106)⋅(106)2=2×10−12×1012=2 rad2⋅Hz.K=S_\phi(10^6)\cdot(10^6)^2=2\times10^{-12}\times10^{12}=2\ \text{rad}^2\cdot\text{Hz}.K=Sϕ​(106)⋅(106)2=2×10−12×1012=2 rad2⋅Hz.

σϕ2=∫f1f2Kf2df=K ⁣(1f1−1f2)=2(1106−1107)=2×9×10−7=1.8×10−6 rad2.\sigma_\phi^2=\int_{f_1}^{f_2}\frac{K}{f^2}df=K\!\left(\frac{1}{f_1}-\frac{1}{f_2}\right)=2\left(\frac{1}{10^6}-\frac{1}{10^7}\right)=2\times9\times10^{-7}=1.8\times10^{-6}\ \text{rad}^2.σϕ2​=∫f1​f2​​f2K​df=K(f1​1​−f2​1​)=2(1061​−1071​)=2×9×10−7=1.8×10−6 rad2.σϕ=1.8×10−6≈1.342×10−3 rad=1.342 mrad.\sigma_\phi=\sqrt{1.8\times10^{-6}}\approx1.342\times10^{-3}\ \text{rad}=1.342\ \text{mrad}.σϕ​=1.8×10−6​≈1.342×10−3 rad=1.342 mrad.

Dimension check：(a) L\mathcal{L}L、SϕS_\phiSϕ​ 都是 per-Hz（rad²/Hz）✓； (b) [K]⋅[1/f]=rad2⋅Hz⋅Hz−1=rad2[K]\cdot[1/f]=\text{rad}^2\cdot\text{Hz}\cdot\text{Hz}^{-1}=\text{rad}^2[K]⋅[1/f]=rad2⋅Hz⋅Hz−1=rad2 ✓；radrad/s=s\dfrac{\text{rad}}{\text{rad/s}}=\text{s}rad/srad​=s ✓。

import numpy as np

L_dbc = -120.0
