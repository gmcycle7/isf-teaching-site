LTI vs LTV — 振盪器對 noise 的敏感度是週期時變的

先備：oscillator_phase · phase_vs_amplitude_noise ｜ 接下來：從 impulse 到 phase shift 的推導

訊號與系統課教我們用脈衝響應 h(t)h(t)h(t) 與卷積描述線性系統。但那套是 LTI（Linear Time-Invariant，線性非時變） 的故事。Hajimiri–Lee 在 [P1] 最關鍵的洞見是：

振盪器對 noise 的相位響應不是 LTI，而是 LTV（Linear Time-Variant，線性時變）。

這一頁說清楚 LTI 與 LTV 差在哪、為什麼振盪器一定是 LTV、以及為什麼這個差別正好就是 ISF（Impulse Sensitivity Function，脈衝敏感度函數） 的存在理由。

物理直覺（先講結論）：LTI 系統「不看現在幾點」——同一個脈衝，今天打跟明天打， 反應形狀一模一樣，只是時間平移。振盪器不是這樣：因為它的狀態在 limit cycle 上不停 轉動，同一顆 noise 電流注在「波峰」跟注在「零交越」會得到完全不同的相位偏移。 系統對輸入的敏感度本身隨時間（隨注入相位）週期變化——這就是「週期時變敏感度」， 把它整理成一個 2π2\pi2π 週期、無因次的函數，就是 ISF Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ)。

1. 複習：LTI 的 h(t−τ)h(t-\tau)h(t−τ)​

線性非時變系統有兩個性質：線性（疊加成立）與時不變（輸入延遲 Δ\DeltaΔ，輸出也只是 延遲 Δ\DeltaΔ、形狀不變）。時不變的後果是它的脈衝響應只依賴時間差 t−τt-\taut−τ：

y(t)=∫−∞∞h(t−τ) x(τ) dτ（LTI 卷積）.y(t)=\int_{-\infty}^{\infty}h(t-\tau)\,x(\tau)\,d\tau\qquad\text{（LTI 卷積）}.y(t)=∫−∞∞​h(t−τ)x(τ)dτ（LTI 卷積）.
  • 為何只看 t−τt-\taut−τ：在 τ1\tau_1τ1​ 注入單位脈衝得到 h(t−τ1)h(t-\tau_1)h(t−τ1​)，在 τ2\tau_2τ2​ 注入得到 h(t−τ2)h(t-\tau_2)h(t−τ2​)——同一個形狀，只是平移。系統不在乎「絕對時刻 τ\tauτ」，只在乎「過了多久」。
  • 單位檢查：卷積核 hhh 的維度由輸出／輸入決定；這裡重點是「形狀不變」而非數值。
  • 這就是大學學的那套：RLC 濾波器、放大器小訊號等效——只要工作點固定，都是 LTI。

2. 振盪器：脈衝響應變成 h(t,τ)h(t,\tau)h(t,τ)，多了一個自變數​

振盪器沒有固定工作點——它的狀態在 limit cycle 上不停移動（見 oscillator_phase）。所以「此刻系統對擾動多敏感」會隨 狀態（也就是隨注入相位 ω0τ\omega_0\tauω0​τ）改變。脈衝響應因此同時依賴兩個時間：

  hϕ(t,τ)≠hϕ(t−τ)  （LTV：依賴絕對注入時刻τ，不只t−τ）.\boxed{\;h_\phi(t,\tau)\neq h_\phi(t-\tau)\;}\qquad\text{（LTV：依賴絕對注入時刻}\tau\text{，不只}t-\tau\text{）}.hϕ​(t,τ)=hϕ​(t−τ)​（LTV：依賴絕對注入時刻τ，不只t−τ）.

[P1] 寫出的 excess-phase 脈衝響應正是這個形式（[P1] Eq.(10), p.182）：

hϕ(t,τ)=Γ(ω0τ)qmax u(t−τ).h_\phi(t,\tau)=\frac{\Gamma(\omega_0\tau)}{q_{max}}\,u(t-\tau).hϕ​(t,τ)=qmax​Γ(ω0​τ)​u(t−τ).

把它拆開看，剛好把 LTV 的兩個自變數分給兩個因子，意義清楚：

  • 階高（step height）Γ(ω0τ)qmax\dfrac{\Gamma(\omega_0\tau)}{q_{max}}qmax​Γ(ω0​τ)​：只依賴注入相位 ω0τ\omega_0\tauω0​τ。 這是「時變」的部分——同樣大小的脈衝，注入相位不同，跳的高度不同。

  • 時間形狀 u(t−τ)u(t-\tau)u(t−τ)：一個 unit step（單位階梯），只依賴 t−τt-\taut−τ。它編碼「相位偏移 一旦發生就永久保持」（相位沒有恢復力，見 phase_vs_amplitude_noise）。

  • 單位檢查 / dimension：Γ\GammaΓ 無因次、qmaxq_{max}qmax​ 單位 C、uuu 無因次，所以 hϕh_\phihϕ​ 的單位是 C−1\mathrm{C^{-1}}C−1；對電流（A）卷積後 ∫hϕ i dτ\int h_\phi\,i\,d\tau∫hϕ​idτ 得 C−1⋅A⋅s=C−1⋅C=\mathrm{C^{-1}\cdot A\cdot s}=\mathrm{C^{-1}\cdot C}=C−1⋅A⋅s=C−1⋅C= 無因次 = rad ✓。

  • 為何是 Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ) 而不是 Γ(τ)\Gamma(\tau)Γ(τ)：敏感度隨「波形相位」週期變化， 週期是 2π2\pi2π（每轉一圈 limit cycle 回到相同敏感度）。所以 Γ\GammaΓ 的自變數是相位 ω0τ\omega_0\tauω0​τ，且 Γ\GammaΓ 是 2π2\pi2π 週期函數——這就是「週期時變敏感度」。

3. 為什麼同一個 impulse 在不同位置得到不同 phase shift​

把第 2 步的 step height 套到操作型 ISF 定義（[P1] Eq.(11) 的脈衝極限）：

Δϕ(τ)=Γ(ω0τ)qmax Δq.\Delta\phi(\tau)=\frac{\Gamma(\omega_0\tau)}{q_{max}}\,\Delta q.Δϕ(τ)=qmax​Γ(ω0​τ)​Δq.

固定 Δq\Delta qΔq、掃注入相位 θ=ω0τ\theta=\omega_0\tauθ=ω0​τ，Δϕ\Delta\phiΔϕ 就照著 Γ(θ)\Gamma(\theta)Γ(θ) 的形狀 起伏。對理想 LC，Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ：

  • 物理原因（接 oscillator_phase 第 4 步）：相位偏移大小取決於電壓跳變投影到 limit cycle 切線方向的分量。零交越時狀態速度（切線）正好沿著電壓軸，固定的 ΔV\Delta VΔV 幾乎 全部變成切向位移 → Δϕ\Delta\phiΔϕ 最大；波峰時切線垂直於電壓軸，ΔV\Delta VΔV 幾乎全是徑向 （純振幅）→ Δϕ≈0\Delta\phi\approx 0Δϕ≈0。切向分量 ∝sin⁡θ\propto\sin\theta∝sinθ，故對正弦波 Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ。
  • 這就是 LTV 的指紋：若振盪器是 LTI，Δϕ\Delta\phiΔϕ 應該與 τ\tauτ 無關（只看 t−τt-\taut−τ）—— 但實際上 Δϕ(τ)\Delta\phi(\tau)Δϕ(τ) 週期起伏。敏感度本身隨時間變 = time-variant。

4. 對照圖：LTI 形狀不變 vs LTV 階高隨相位變​

下圖上半是一個 LTI 系統（簡單一階衰減）的脈衝響應：在 τ=0.3, 0.9, 1.5\tau=0.3,\,0.9,\,1.5τ=0.3,0.9,1.5 注入， 得到同一個形狀、只是平移。下半是振盪器的 LTV 相位響應 hϕ(t,τ)h_\phi(t,\tau)hϕ​(t,τ)：在不同注入 相位 τ\tauτ，階梯高度 Γ(ω0τ)=−sin⁡(2πτ)\Gamma(\omega_0\tau)=-\sin(2\pi\tau)Γ(ω0​τ)=−sin(2πτ) 跟著改變（甚至變號）。

這張圖怎麼解讀：

  • 上半（LTI）：三條曲線疊起來形狀一模一樣，只是起點平移——「系統不看絕對時刻」。
  • 下半（LTV）：三條都是階梯（u(t−τ)u(t-\tau)u(t−τ)，永久保持），但階高不同：τ=0\tau=0τ=0 時 Γ=−sin⁡0=0\Gamma=-\sin 0=0Γ=−sin0=0（注在波峰，階高 0、無相位偏移）；τ=0.25\tau=0.25τ=0.25 時 Γ=−sin⁡(π/2)=−1\Gamma=-\sin(\pi/2)=-1Γ=−sin(π/2)=−1 （注在零交越，階高最大、負向）；τ=0.5\tau=0.5τ=0.5 時 Γ=−sin⁡(π)=0\Gamma=-\sin(\pi)=0Γ=−sin(π)=0（注在波谷，階高 0）。
  • 訊息：LTV 的「形狀」固定（都是階梯，因相位無恢復力），但「強度」週期時變——這正是 ISF。

5. 為什麼非得用 LTV——LTI 模型錯在哪​

歷史上早期把振盪器 noise 當 LTI（如把雜訊直接乘上 tank 的轉移函數）會漏掉兩件事，正是 LTV／ISF 才補得起來的：

  1. 頻率轉換（frequency translation）：因為敏感度 Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ) 本身是週期的 （含 ω0\omega_0ω0​ 的諧波），它會把 0, ω0, 2ω0,…0,\,\omega_0,\,2\omega_0,\dots0,ω0​,2ω0​,… 附近的 noise 混頻搬到 載波旁邊。LTI 不會混頻，因此無法解釋為何 device 在 DC 與在 nω0n\omega_0nω0​ 的 noise 都會出現在 close-in 相位雜訊裡。這由 ISF 的傅立葉係數 cnc_ncn​ 描述（[P1] Eq.(12)–(13), p.183）。
  2. 1/f1/f1/f 上轉成 1/f31/f^31/f3：device 的低頻 1/f1/f1/f noise 之所以能搬到載波附近，靠的是 ISF 的 DC 係數 c0c_0c0​（[P1] Eq.(23)），純 LTI 觀點解釋不了。波形對稱 → c0c_0c0​ 小 → 1/f31/f^31/f3 角 被壓低（[P1] Eq.(24)）。

換句話說，LTV 不是把問題複雜化，而是因為振盪器物理上就是時變的；強行套 LTI 會在定性上 就漏掉混頻與 1/f1/f1/f 上轉這兩個真實現象。

數值例子（建立手感）​

例 A 的 LTV 版：qmax=1q_{max}=1qmax​=1 pC、Δq=1\Delta q=1Δq=1 fC、f0=5f_0=5f0​=5 GHz，比較注在零交越 （Γ=−1\Gamma=-1Γ=−1）與注在峰值（Γ=0\Gamma=0Γ=0）。

重點回顧​

  • LTI：h(t−τ)h(t-\tau)h(t−τ)，只看時間差，換注入時刻只是平移、形狀不變。
  • LTV：hϕ(t,τ)=Γ(ω0τ)qmaxu(t−τ)h_\phi(t,\tau)=\dfrac{\Gamma(\omega_0\tau)}{q_{max}}u(t-\tau)hϕ​(t,τ)=qmax​Γ(ω0​τ)​u(t−τ)——階高隨注入相位 ω0τ\omega_0\tauω0​τ 週期變化，時間形狀（階梯）固定。
  • 振盪器一定是 LTV：狀態沿 limit cycle 移動 → 敏感度隨相位變 → 同 impulse 不同位置 → 不同 Δϕ\Delta\phiΔϕ。
  • ISF Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ) 的本質就是「週期時變敏感度」；理想 LC 為 −sin⁡θ-\sin\theta−sinθ。
  • LTV 才解釋得了頻率轉換（cnc_ncn​ 混頻）與 1/f→1/f31/f\to1/f^31/f→1/f3 上轉（c0c_0c0​），LTI 漏掉這兩者。
  • 例 A：1 fC 注零交越 → 31.8 fs；注波峰 → 0 fs（敏感度週期時變的直接後果）。
  • 來源：[P1] Eq.(10), Eq.(11), Sec. III；數值與圖出自 lab_04。
