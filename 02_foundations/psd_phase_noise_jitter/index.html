Phase Noise → Jitter

先備：white_noise_to_phase_noise · stochastic_noise_basics ｜ 接下來：serdes_clocking_connection

這頁回答一個工程上最常被問的問題：手上拿到一張 phase noise 圖（L(f)\mathcal{L}(f)L(f)，dBc/Hz）， 怎麼把它變成一個時域數字——rms jitter σt\sigma_tσt​（fs）？ 這是 frequency-domain（頻域， 通訊/RF 的語言）與 time-domain（時域，數位/SerDes 的語言）之間的橋。

整條鏈是四步：

L(f) → ×2, de-dB  Sϕ(f) → ∫f1f2  σϕ2 →    σϕ → ÷(2πf0)  σt.\mathcal{L}(f)\ \xrightarrow{\ \times2,\ \text{de-dB}\ }\ S_\phi(f)\ \xrightarrow{\ \int_{f_1}^{f_2}\ }\ \sigma_\phi^2\ \xrightarrow{\ \sqrt{\ }\ }\ \sigma_\phi\ \xrightarrow{\ \div(2\pi f_0)\ }\ \sigma_t.L(f)  ×2, de-dB ​ Sϕ​(f)  ∫f1​f2​​ ​ σϕ2​   ​ ​ σϕ​  ÷(2πf0​) ​ σt​.

我們把每一步都拆開、帶單位、帶 dimension check，最後用 canonical 例 C （5 GHz、−100-100−100 dBc/Hz @ 1 MHz、1/f² 斜率、1→100 MHz）算出 σt=447.9\sigma_t=447.9σt​=447.9 fs。

物理直覺（先講結論）：相位誤差 Δϕ\Delta\phiΔϕ 就是「時鐘的指針偏了多少角度」； 把角度除以角速度 2πf02\pi f_02πf0​，就得到「指針偏了多少時間」Δt\Delta tΔt。phase noise 圖 告訴你每個 offset 頻率有多少相位功率密度；把它們**全部加起來（積分）**就是總相位 變異數；開根號是 rms 相位；再除 2πf02\pi f_02πf0​ 就是 rms timing jitter。整張圖被壓成一個 fs 數字。

第 1 步：phase error 為什麼能轉成 timing error​

一個理想振盪 cos⁡(2πf0t)\cos(2\pi f_0 t)cos(2πf0​t)，加上 excess phase 後是 cos⁡(2πf0t+Δϕ)\cos(2\pi f_0 t+\Delta\phi)cos(2πf0​t+Δϕ)。 把相位項提出來看 zero crossing（過零點）落在哪：

2πf0t+Δϕ=2πf0(t+Δϕ2πf0⏟= Δt).2\pi f_0 t+\Delta\phi=2\pi f_0\Big(t+\underbrace{\frac{\Delta\phi}{2\pi f_0}}_{=\ \Delta t}\Big).2πf0​t+Δϕ=2πf0​(t+= Δt2πf0​Δϕ​​​).

也就是說，多出 Δϕ\Delta\phiΔϕ 的相位，等效於整條波形在時間軸上平移了 Δt\Delta tΔt （規範第 3 節公式 17）：

 Δt=Δϕ2πf0 \boxed{\ \Delta t=\frac{\Delta\phi}{2\pi f_0}\ } Δt=2πf0​Δϕ​ ​
  • 用到的物理：phase 與 time 之間的轉換率就是角頻率 ω0=2πf0\omega_0=2\pi f_0ω0​=2πf0​（rad/s）。
  • dimension check：[rad]/[rad/s]=[s][\text{rad}]/[\text{rad/s}]=[\text{s}][rad]/[rad/s]=[s] ✓。注意分母是 2πf02\pi f_02πf0​ （rad/s），不是 f0f_0f0​（Hz）——這個 2π2\pi2π 漏掉是最常見的錯。
  • 為何合理：在小相位（Δϕ≪1\Delta\phi\ll1Δϕ≪1 rad）下，相位偏移與邊緣時間偏移是線性、 一對一的。timing jitter 就是 zero crossing 的時間誤差，所以它 = Δϕ/(2πf0)\Delta\phi/(2\pi f_0)Δϕ/(2πf0​)。
  • 手感：5 GHz 下 Δϕ=1\Delta\phi=1Δϕ=1 mrad ⇒Δt=31.8\Rightarrow\Delta t=31.8⇒Δt=31.8 fs（見 numerical_feeling Example 1）。

第 2 步：dBc/Hz 怎麼轉 linear，並還原 phase PSD​

phase noise 圖的縱軸是 L(f)\mathcal{L}(f)L(f)，SSB phase noise（single-sideband， 單邊帶相位雜訊），單位 dBc/Hz——意思是「在 offset fff 處、每 1 Hz 頻寬內， 單邊帶的雜訊功率比 carrier 低幾 dB」（dBc = dB relative to carrier）。

de-dB（從 dB 換回線性）：dBc/Hz 是 10log⁡10(⋅)10\log_{10}(\cdot)10log10​(⋅)，所以

Llin(f)=10L(f)/10[1/Hz].\mathcal{L}_{\text{lin}}(f)=10^{\mathcal{L}(f)/10}\quad[\text{1/Hz}].Llin​(f)=10L(f)/10[1/Hz].

第 3 步：為什麼對 phase PSD 積分能得到 variance​

  • dimension check：(rad2/Hz)×Hz=rad2(\text{rad}^2/\text{Hz})\times\text{Hz}=\text{rad}^2(rad2/Hz)×Hz=rad2 ✓。
  • 為什麼積分頻寬 [f1,f2][f_1,f_2][f1​,f2​] 很重要：phase PSD 在低 offset 通常是 1/f²（甚至 1/f³）， 積分量會被下限 f1f_1f1​ 主導。改 f1f_1f1​ 一個 decade，jitter 可能差好幾倍。所以報 jitter 一定要標積分頻寬，否則數字沒有意義。
    • 上限 f2f_2f2​：物理上由系統頻寬決定（SerDes 是 PLL loop bandwidth 或 Nyquist）。
    • 下限 f1f_1f1​：對開環振盪器積到 DC 會發散（random walk），實務上由量測時間或 PLL 把它「拉住」的頻率決定。
  • 小角近似（sin⁡Δϕ≈Δϕ\sin\Delta\phi\approx\Delta\phisinΔϕ≈Δϕ）：把相位調變的功率近似成 σϕ2\sigma_\phi^2σϕ2​， 同樣要求 σϕ≪1\sigma_\phi\ll1σϕ​≪1 rad。

f  = np.logspace(6, 8, 200000)
