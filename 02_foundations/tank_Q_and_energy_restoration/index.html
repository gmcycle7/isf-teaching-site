Tank Q 與能量恢復

「品質因數 QQQ（quality factor，諧振有多『尖』、每週期能量被耗掉多少的指標）」這個量幾乎在本站每一頁都出現：Leeson 模型寫成 12Q\dfrac{1}{2Q}2Q1​（derivation_leeson）、LC vs ring 的比較靠它（lc_vs_ring）、tank swing 的取捨也要它（tank_swing）。可是它在那些頁裡都被當成已知量直接用，從來沒有從電路被乾淨推出來過。這一頁就把這個洞補上：從最基本的並聯 RLC tank 出發，把 QQQ 的三種等價寫法與能量定義一步步推出來，再說明它如何接到 active core 的 −R-R−R、tank 熱雜訊 4kT/Rp4kT/R_p4kT/Rp​，以及最後如何決定 phase noise。

物理直覺（先講結論）：tank（諧振槽，LLL 與 CCC 來回交換能量的儲能元件）就像一個鐘擺。QQQ 衡量「這個鐘擺有多不願意停下來」——它每振盪一個 radian，相對於儲存的能量只漏掉一小撮給損耗電阻 RpR_pRp​。QQQ 越高，諧振峰越尖、頻寬越窄、相位對頻率的斜率越陡。把 noise 想成想把振盪頻率推歪的力：QQQ 越高，tank 越「咬死」在 ω0\omega_0ω0​、越不肯被推歪，於是同一坨 noise 換來的 phase noise 越小。這就是為什麼低 phase noise 設計的第一句口號永遠是「把 QQQ 做高」。

這頁要回答三件事：

  1. QQQ 到底是什麼？為什麼 Q=RpC/L=ω0RpC=Rp/(ω0L)Q=R_p\sqrt{C/L}=\omega_0 R_p C=R_p/(\omega_0 L)Q=Rp​C/L​=ω0​Rp​C=Rp​/(ω0​L) 三種寫法等價，又為什麼它等於 ω0×\omega_0\timesω0​×（儲能/耗能功率）？
  2. 真實 tank 一定有損耗 RpR_pRp​，振盪會衰減——active core 怎麼用 −R-R−R 把它補回來？而那個被補掉的 RpR_pRp​，為什麼正是 tank 熱雜訊 4kT/Rp4kT/R_p4kT/Rp​ 的源頭？
  3. QQQ 怎麼接到 phase noise？為什麼「高 QQQ → 窄帶 → 陡相位斜率 → 每單位 offset 的 noise 更少」，而這跟 ISF 的 Γrms/qmax\Gamma_{rms}/q_{max}Γrms​/qmax​ 是同一回事？

誠實聲明（請先讀）：本頁的 RLC tank、QQQ 的定義與三種等價寫法、4kTR4kTR4kTR 熱雜訊都是標準電路學/微波工程教科書內容（外部，非本站下載的 5 篇 PDF），例如 Razavi《RF Microelectronics》、Pozar《Microwave Engineering》、Lee《The Design of CMOS RFICs》。本站不重新發明這些常數，只把它們乾淨推一遍，並接到 5 篇 PDF 內已核的 ISF 結果（[P1] Eq.(21) 等）。凡是接到 [P1]/[P2] 的地方都有標明 paper id + equation。

第 1 步：並聯 RLC tank 與 QQQ 的三種等價寫法​

最基本的 LC 振盪器 tank 是一個並聯 RLC：電感 LLL、電容 CCC、以及一個並聯損耗電阻 RpR_pRp​（把所有 tank 損耗——inductor 串聯電阻、capacitor 介質損耗、輻射——全部等效成這一個並聯電阻）。

先定義諧振角頻率。並聯 LC 在某個頻率讓電感與電容的電納（susceptance）相消、阻抗純電阻、能量在 LLL 與 CCC 之間完全來回交換：

ω0=1LC.\omega_0=\frac{1}{\sqrt{LC}}.ω0​=LC​1​.
  • 用到的物理：電感阻抗 jωLj\omega LjωL、電容阻抗 1/(jωC)1/(j\omega C)1/(jωC)。並聯時電納相加 1jωL+jωC=j(ωC−1ωL)\dfrac{1}{j\omega L}+j\omega C=j\big(\omega C-\dfrac{1}{\omega L}\big)jωL1​+jωC=j(ωC−ωL1​)，令括號為零即得 ω02=1/(LC)\omega_0^2=1/(LC)ω02​=1/(LC)。
  • 單位檢查：[LC]=H⋅F=(V⋅s/A)(A⋅s/V)=s2=s[\sqrt{LC}]=\sqrt{\text{H}\cdot\text{F}}=\sqrt{(\text{V·s/A})(\text{A·s/V})}=\sqrt{\text{s}^2}=\text{s}[LC​]=H⋅F​=(V⋅s/A)(A⋅s/V)​=s2​=s，所以 1/LC1/\sqrt{LC}1/LC​ 是 rad/s ✓。

unloaded（無載）vs loaded（有載）QQQ 的區別先講清楚：

  • unloaded QQQ（Q0Q_0Q0​，無載品質因數）：只算 tank 自身損耗（RpR_pRp​ 純粹是 tank 元件的損耗）時的 QQQ。這是 tank 元件本身的「品質」。
  • loaded QQQ（QLQ_LQL​，有載品質因數）：把外部負載（接出去的電路、量測儀器、buffer 輸入阻抗）也並進來後的 QQQ。外部負載相當於再並一個電阻 RextR_{ext}Rext​，總並聯電阻 Rp∥RextR_p\parallel R_{ext}Rp​∥Rext​ 比 RpR_pRp​ 小，所以 QL<Q0Q_L<Q_0QL​<Q0​。
  • 兩者關係：1QL=1Q0+1Qext\dfrac{1}{Q_L}=\dfrac{1}{Q_0}+\dfrac{1}{Q_{ext}}QL​1​=Q0​1​+Qext​1​（電導相加 → QQQ 倒數相加，因為 Q∝Rp∝1/GQ\propto R_p\propto 1/GQ∝Rp​∝1/G）。本頁推導與決定 phase noise 的，主要是 loaded QQQ（振盪器實際看到的）；但設計 inductor 時關心的是 unloaded QQQ。下文除非特別說明，QQQ 指振盪迴路實際的有效（loaded）QQQ。

現在推 QQQ。並聯 RLC 的 QQQ 的電路定義是「諧振時，電抗元件（LLL 或 CCC）儲存的能量流，相對於電阻 RpR_pRp​ 耗散的速率」。最方便的等價代數定義是：諧振時電容（或電感）的電納大小對電導的比值

Q=∣BC(ω0)∣G=ω0C1/Rp=ω0RpC.Q=\frac{|B_C(\omega_0)|}{G}=\frac{\omega_0 C}{1/R_p}=\omega_0 R_p C.Q=G∣BC​(ω0​)∣​=1/Rp​ω0​C​=ω0​Rp​C.
  • 用到的物理：並聯 RLC 中，電導 G=1/RpG=1/R_pG=1/Rp​ 是唯一耗能元件；電容電納 BC=ω0CB_C=\omega_0 CBC​=ω0​C 量「無功電流」有多大。QQQ 就是無功電流 / 有功電流的比。
  • 單位檢查：[ω0RpC]=(rad/s)(Ω)(F)=(1/s)(V/A)(A⋅s/V)=[\omega_0 R_p C]=(\text{rad/s})(\Omega)(\text{F})=(\text{1/s})(\text{V/A})(\text{A·s/V})=[ω0​Rp​C]=(rad/s)(Ω)(F)=(1/s)(V/A)(A⋅s/V)= 無因次 ✓。

把 ω0=1/LC\omega_0=1/\sqrt{LC}ω0​=1/LC​ 代進去，得到第二種寫法：

Q=ω0RpC=RpCLC=RpC2LC=RpCL.Q=\omega_0 R_p C=\frac{R_p C}{\sqrt{LC}}=R_p\sqrt{\frac{C^2}{LC}}=R_p\sqrt{\frac{C}{L}}.Q=ω0​Rp​C=LC​Rp​C​=Rp​LCC2​​=Rp​LC​​.

  • 單位檢查：C/L=F/H=(A⋅s/V)/(V⋅s/A)=A2/V2=A/V=1/Ω\sqrt{C/L}=\sqrt{\text{F/H}}=\sqrt{(\text{A·s/V})/(\text{V·s/A})}=\sqrt{\text{A}^2/\text{V}^2}=\text{A/V}=1/\OmegaC/L​=F/H​=(A⋅s/V)/(V⋅s/A)​=A2/V2​=A/V=1/Ω，乘 RpR_pRp​（Ω\OmegaΩ）→ 無因次 ✓。L/C\sqrt{L/C}L/C​ 這個量有電阻的單位，叫 tank 的特性阻抗（characteristic impedance） R0=L/CR_0=\sqrt{L/C}R0​=L/C​，所以也可寫成 Q=Rp/R0Q=R_p/R_0Q=Rp​/R0​——「並聯損耗電阻相對特性阻抗有多大」。

第三種寫法用 ω0=1/LC\omega_0=1/\sqrt{LC}ω0​=1/LC​ 把 CCC 換成 LLL。因為諧振時電感電納大小等於電容電納大小（ω0L\omega_0 Lω0​L 與 1/(ω0C)1/(\omega_0 C)1/(ω0​C) 互為倒數），所以同一個比值也可寫成電導 ×\times× 電感電抗的倒數：

第 3 步：active core 補 −R-R−R 抵消 RpR_pRp​，而 RpR_pRp​ 正是 4kT/Rp4kT/R_p4kT/Rp​ 熱雜訊的源頭​

第 2 步說明了：只要有 RpR_pRp​，振盪就會以 e−ω0t/(2Q)e^{-\omega_0 t/(2Q)}e−ω0​t/(2Q) 衰減。要維持穩定振盪，振盪器的 active core（主動核心，例如 cross-coupled 差動對提供的負電導） 必須把 RpR_pRp​ 漏掉的能量逐週期補回來。最乾淨的看法：active core 在 tank 兩端呈現一個負電阻 −R-R−R（負電導 −Gm-G_m−Gm​），與 RpR_pRp​ 並聯。

1Rtank=1Rp−Gm,起振條件: Gm≥1Rp (即 −R≤−Rp).\frac{1}{R_{tank}}=\frac{1}{R_p}-G_m,\qquad \text{起振條件}:\ G_m\ge\frac{1}{R_p}\ \big(\text{即 } -R\le -R_p\big).Rtank​1​=Rp​1​−Gm​,起振條件: Gm​≥Rp​1​ (即 −R≤−Rp​).
  • 物理意義：正電阻吸能、負電阻供能。當 active core 的負電導 GmG_mGm​ 剛好等於 RpR_pRp​ 的電導 1/Rp1/R_p1/Rp​，總損耗為零，振盪維持等幅（Barkhausen 起振邊界）。實務上設計 GmG_mGm​ 略大於 1/Rp1/R_p1/Rp​（loop gain >1>1>1）保證能起振，再靠非線性飽和把有效 GmG_mGm​ 拉回到剛好抵消——這就是 oscillator_phase 講的 limit cycle 的振幅恢復機制在電路層的化身。
  • 單位檢查：GmG_mGm​ 與 1/Rp1/R_p1/Rp​ 都是西門子（S）✓。
  • 關鍵釐清：active core 抵消的是 RpR_pRp​ 的確定性能量損耗（讓振盪不衰減）。它不會、也不能消掉 RpR_pRp​ 帶來的隨機熱雜訊——這是下一段的重點，也是整頁與 phase noise 接軌的橋。

RpR_pRp​ 是 tank 熱雜訊電流 4kT/Rp4kT/R_p4kT/Rp​ 的物理來源。 任何耗能電阻（包括等效損耗 RpR_pRp​）依 Johnson–Nyquist 定理一定伴隨熱雜訊。並聯電阻最方便用諾頓等效的雜訊電流源表示，其單邊 PSD：

in,R2‾Δf=4kTRp.\frac{\overline{i_{n,R}^2}}{\Delta f}=\frac{4kT}{R_p}.Δfin,R2​​​=Rp​4kT​.
  • 用到的物理：fluctuation–dissipation（漲落–耗散）定理——有耗散就有漲落。電阻熱雜訊電壓 PSD 是 4kTRp4kTR_p4kTRp​（psd_phase_noise_jitter 有講），換成並聯諾頓電流源就是除以 Rp2R_p^2Rp2​：in2‾/Δf=(4kTRp)/Rp2=4kT/Rp\overline{i_n^2}/\Delta f=(4kTR_p)/R_p^2=4kT/R_pin2​​/Δf=(4kTRp​)/Rp2​=4kT/Rp​。
  • 單位檢查：[4kT/Rp]=J/Ω=(V⋅A⋅s)/(V/A)=A2s=A2/Hz[4kT/R_p]=\text{J}/\Omega=(\text{V·A·s})/(\text{V/A})=\text{A}^2\text{s}=\text{A}^2/\text{Hz}[4kT/Rp​]=J/Ω=(V⋅A⋅s)/(V/A)=A2s=A2/Hz ✓（單邊電流 PSD，符合規範 notation 的 in2‾/Δf\overline{i_n^2}/\Delta fin2​​/Δf 單位 A²/Hz）。
  • 深刻之處：你沒辦法用 −R-R−R 把這個雜訊也消掉。active core 的 −R-R−R 抵消的是 RpR_pRp​ 在「能量平衡」帳上的損耗項；但 RpR_pRp​ 在「漲落」帳上注入的 4kT/Rp4kT/R_p4kT/Rp​ 隨機電流，被 active core 補進來的能量同等放大維持。所以tank 損耗 RpR_pRp​ 同時是兩件事的源頭：它讓你需要 active core 補能，也讓你拿到一個無法迴避的基礎熱雜訊地板。這正是把 QQQ 與 phase noise 綁在一起的物理根。（active core 自己的 device 還會再加雜訊——那是 device_noise_mapping 的主題；本頁只談 tank 本身的 4kT/Rp4kT/R_p4kT/Rp​。）

把 RpR_pRp​ 用 QQQ 表達（之後好接 phase noise）：由第 1 步 Rp=Q/(ω0C)=Q ω0L=Q R0R_p=Q/(\omega_0 C)=Q\,\omega_0 L=Q\,R_0Rp​=Q/(ω0​C)=Qω0​L=QR0​，所以同樣電容/頻率下，

  • 讀法：QQQ 越高 → RpR_pRp​ 越大 → tank 注入的熱雜訊電流 PSD 越小（∝1/Q\propto 1/Q∝1/Q）。這是「高 QQQ → 低 phase noise」的第一層原因（雜訊源變小）；第 2 步的「窄帶/陡斜率」是第二層。兩層疊起來就是下一步。
