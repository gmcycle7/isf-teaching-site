習題 6 解答（Fourier 係數的頻率搬移意義 — 推導）

目標式。 第 nnn 諧波項的相位貢獻（[P1] Eq.(13)）：

ϕn(t)=1qmax∫t ⁣ ⁣cncos⁡(nω0τ+θn) I0cos⁡((nω0+Δω)τ) dτ.\phi_n(t)=\frac{1}{q_{max}}\int^{t}\!\!c_n\cos(n\omega_0\tau+\theta_n)\,I_0\cos\big((n\omega_0+\Delta\omega)\tau\big)\,d\tau.ϕn​(t)=qmax​1​∫tcn​cos(nω0​τ+θn​)I0​cos((nω0​+Δω)τ)dτ.

第 (i) 步：積化和差。 令 A=nω0τ+θnA=n\omega_0\tau+\theta_nA=nω0​τ+θn​、B=(nω0+Δω)τB=(n\omega_0+\Delta\omega)\tauB=(nω0​+Δω)τ，用 cos⁡Acos⁡B=12[cos⁡(A−B)+cos⁡(A+B)]\cos A\cos B=\tfrac12[\cos(A-B)+\cos(A+B)]cosAcosB=21​[cos(A−B)+cos(A+B)]：

A−B=θn−Δωτ,A+B=(2nω0+Δω)τ+θn.A-B=\theta_n-\Delta\omega\tau,\qquad A+B=(2n\omega_0+\Delta\omega)\tau+\theta_n.A−B=θn​−Δωτ,A+B=(2nω0​+Δω)τ+θn​.

被積函數 =I0cn2[cos⁡(Δωτ−θn)⏟慢，≈Δω+cos⁡((2nω0+Δω)τ+θn)⏟快，≈2nω0]=\dfrac{I_0c_n}{2}\Big[\underbrace{\cos(\Delta\omega\tau-\theta_n)}_{\text{慢，}\approx\Delta\omega}+\underbrace{\cos((2n\omega_0+\Delta\omega)\tau+\theta_n)}_{\text{快，}\approx2n\omega_0}\Big]=2I0​cn​​[慢，≈Δωcos(Δωτ−θn​)​​+快，≈2nω0​cos((2nω0​+Δω)τ+θn​)​​]。

第 (ii) 步：積分器是低通，快項被平均掉。

  • 慢項：∫cos⁡(Δωτ−θn)dτ=sin⁡(Δωτ−θn)Δω\int\cos(\Delta\omega\tau-\theta_n)d\tau=\dfrac{\sin(\Delta\omega\tau-\theta_n)}{\Delta\omega}∫cos(Δωτ−θn​)dτ=Δωsin(Δωτ−θn​)​——分母只有小小的 Δω\Delta\omegaΔω，被放大、存活。
  • 快項：sin⁡(⋯ )2nω0+Δω\dfrac{\sin(\cdots)}{2n\omega_0+\Delta\omega}2nω0​+Δωsin(⋯)​——分母是巨大的 2nω02n\omega_02nω0​，幅度被壓到 ∼Δω/(2nω0)\sim\Delta\omega/(2n\omega_0)∼Δω/(2nω0​) 倍，可忽略。

第 (iii) 步：只留慢項。

ϕn(t)≈1qmax⋅I0cn2⋅sin⁡(Δωt−θn)Δω=I0cn2qmax⋅sin⁡(Δωt−θn)Δω.■\phi_n(t)\approx\frac{1}{q_{max}}\cdot\frac{I_0c_n}{2}\cdot\frac{\sin(\Delta\omega t-\theta_n)}{\Delta\omega}=\frac{I_0 c_n}{2q_{max}}\cdot\frac{\sin(\Delta\omega t-\theta_n)}{\Delta\omega}.\qquad\blacksquareϕn​(t)≈qmax​1​⋅2I0​cn​​⋅Δωsin(Δωt−θn​)​=2qmax​I0​cn​​⋅Δωsin(Δωt−θn​)​.■

物理意義（mixer 觀點）：ISF 的第 nnn 諧波 cnc_ncn​ 像一個 LO（本地振盪）梳齒，把 注入頻率 nω0+Δωn\omega_0+\Delta\omeganω0​+Δω 處的 noise 下變頻到 baseband 的 Δω\Delta\omegaΔω 處； 快項（和頻 ≈2nω0\approx2n\omega_0≈2nω0​）被積分器這個低通濾掉。振盪器本身就是一個對自己各諧波取樣的 mixer—— 這就是 cnc_ncn​ 為什麼是「相位輸出對各諧波的轉換係數」（連 fourier_series_of_isf）。

Dimension check：ϕn=I0cn2qmax⋅sin⁡(⋯ )Δω\phi_n=\dfrac{I_0 c_n}{2q_{max}}\cdot\dfrac{\sin(\cdots)}{\Delta\omega}ϕn​=2qmax​I0​cn​​⋅Δωsin(⋯)​ 的單位為 [A]⋅(無因次)[C]⋅[rad/s]\dfrac{[\text{A}]\cdot(\text{無因次})}{[\text{C}]\cdot[\text{rad/s}]}[C]⋅[rad/s][A]⋅(無因次)​；以 C=A⋅s\text{C}=\text{A·s}C=A⋅s、rad 視為無因次代入， =A(A⋅s)⋅(1/s)=AA=1=\dfrac{\text{A}}{(\text{A·s})\cdot(1/\text{s})}=\dfrac{\text{A}}{\text{A}}=1=(A⋅s)⋅(1/s)A​=AA​=1（無因次），故 ϕ\phiϕ 無因次（相位以 rad 計）✓。

import numpy as np
# 數值驗證：直接做積分，看慢項存活、快項消失
n, w0, dw, c_n, I0, qmax, th_n = 1, 1.0, 1e-3, 1.0, 1.0, 1.0, 0.4

習題 7 解答（effective ISF，cyclostationary）

逐步代入。 effective ISF（[P1] Eqs.(25)–(27)）Γeff(θ)=Γ(θ) α(θ)\Gamma_{eff}(\theta)=\Gamma(\theta)\,\alpha(\theta)Γeff​(θ)=Γ(θ)α(θ)。 這裡 α\alphaα 是方波閘控（半週導通），故

Γeff(θ)={−sin⁡θ,θ∈[0,π)0,θ∈[π,2π).\Gamma_{eff}(\theta)=\begin{cases}-\sin\theta,&\theta\in[0,\pi)\\[2pt]0,&\theta\in[\pi,2\pi).\end{cases}Γeff​(θ)={−sinθ,0,​θ∈[0,π)θ∈[π,2π).​

求 rms：

Γeff,rms2=12π∫02πΓeff2 dθ=12π∫0πsin⁡2θ dθ=12π⋅π2=14.\Gamma_{eff,rms}^2=\frac{1}{2\pi}\int_0^{2\pi}\Gamma_{eff}^2\,d\theta=\frac{1}{2\pi}\int_0^{\pi}\sin^2\theta\,d\theta=\frac{1}{2\pi}\cdot\frac{\pi}{2}=\frac14.Γeff,rms2​=2π1​∫02π​Γeff2​dθ=2π1​∫0π​sin2θdθ=2π1​⋅2π​=41​.Γeff,rms=12=0.5.\Gamma_{eff,rms}=\frac12=0.5.Γeff,rms​=21​=0.5.

比較：全程導通 Γrms=1/2≈0.707\Gamma_{rms}=1/\sqrt2\approx0.707Γrms​=1/2​≈0.707；半週閘控後 Γeff,rms=0.5\Gamma_{eff,rms}=0.5Γeff,rms​=0.5。 比值 0.5/0.707=1/20.5/0.707=1/\sqrt20.5/0.707=1/2​——閘掉一半相位，rms 降 2\sqrt22​ 倍（功率降一半）。

結果：Γeff,rms=0.5\Gamma_{eff,rms}=0.5Γeff,rms​=0.5（比全程的 0.7070.7070.707 低 ≈3\approx3≈3 dB 功率）。
