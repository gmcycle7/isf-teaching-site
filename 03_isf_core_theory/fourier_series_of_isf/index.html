ISF 的 Fourier series（傅立葉級數）

前置閱讀：isf_definition（Γ\GammaΓ 是無因次、2π2\pi2π 週期函數）、impulse_to_phase_shift（Γ\GammaΓ 的操作型定義）、convolution_derivation（連續 noise 的相位積分式）。

動手驗證：本頁的 Fourier 係數萃取、重建與 Parseval 數值驗證見 lab_05。

上一章 impulse_to_phase_shift 推出了 ISF 的操作型定義 Δϕ=Γ(ω0τ) Δq/qmax\Delta\phi=\Gamma(\omega_0\tau)\,\Delta q/q_{max}Δϕ=Γ(ω0​τ)Δq/qmax​，並指出 Γ\GammaΓ 是一個 無因次、2π2\pi2π 週期的函數。這頁回答下一個關鍵問題：

既然 Γ\GammaΓ 是週期函數，把它展成 Fourier series 之後，每一項各自代表什麼物理？ 答案是 ISF 理論最漂亮的一段——它把「振盪器把哪一段頻率的 device noise 搬到 carrier 附近」 講得清清楚楚。核心結果（[P1] Eq.(12), p.183）：

Γ(ω0τ)=c02+∑n=1∞cncos⁡(nω0τ+θn)\Gamma(\omega_0\tau)=\frac{c_0}{2}+\sum_{n=1}^{\infty}c_n\cos(n\omega_0\tau+\theta_n)Γ(ω0​τ)=2c0​​+n=1∑∞​cn​cos(nω0​τ+θn​)

物理直覺（先講結論）：振盪器是一台時變的混頻器（mixer）。它對 noise 的相位敏感度 Γ\GammaΓ 隨波形相位週期性變化，等於對 noise 乘上一個以 ω0\omega_0ω0​ 為基頻的「週期權重」。 一個週期函數乘上 noise，數學上就是把 noise 頻譜搬到 0, ω0, 2ω0,…0,\ \omega_0,\ 2\omega_0,\dots0, ω0​, 2ω0​,… 各處再疊加。 所以落在每個 nω0n\omega_0nω0​ 附近的 device noise，都會被第 nnn 個 Fourier 係數 cnc_ncn​ 加權、 然後 down-convert（下轉）到 carrier 附近變成 phase noise。其中 c0c_0c0​（DC 項）特別重要： 它把接近 DC 的 1/f flicker noise 直接搬上來，造成 close-in 的 1/f31/f^31/f3 phase noise。

第 1 步：為什麼 Γ\GammaΓ 可以展成 Fourier series​

Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ) 描述「在波形的某個相位踢一下，相位被推多少」。振盪器處於週期穩態 （periodic steady state，輸出波形每 T=1/f0T=1/f_0T=1/f0​ 重複一次），所以「在相位 xxx 踢」與「在相位 x+2πx+2\pix+2π 踢」效果完全一樣。換言之 Γ\GammaΓ 對自變數 x≡ω0τx\equiv\omega_0\taux≡ω0​τ 是 2π2\pi2π 週期：

Γ(x+2π)=Γ(x).\Gamma(x+2\pi)=\Gamma(x).Γ(x+2π)=Γ(x).
  • 用到的數學：任何 2π2\pi2π 週期、在一個週期內平方可積（∫02π∣Γ∣2dx<∞\int_0^{2\pi}|\Gamma|^2dx<\infty∫02π​∣Γ∣2dx<∞） 的函數都能展成 Fourier series（Dirichlet 條件）。實際 ISF 是連續、分段平滑的，條件輕鬆滿足。
  • 物理依據：週期性來自振盪器的 limit cycle（極限環）——穩態下狀態點每週期繞環一圈， 敏感度只跟「環上的位置（相位）」有關，與「第幾圈」無關。
  • 單位檢查：x=ω0τx=\omega_0\taux=ω0​τ 是 [rad/s]⋅[s]=[rad][\text{rad/s}]\cdot[\text{s}]=[\text{rad}][rad/s]⋅[s]=[rad] ✓，無因次； Γ\GammaΓ 本身無因次，c0,cnc_0,c_nc0​,cn​ 也無因次（見 notation）。

第 2 步：寫出標準 Fourier 展開（cos/sin 形式）​

先用最熟悉的 cos/sin 實數 Fourier series 寫一遍，再合併成 [P1] 的 amplitude–phase 形式。 對 2π2\pi2π 週期函數：

Γ(x)=a02+∑n=1∞[ancos⁡(nx)+bnsin⁡(nx)],\Gamma(x)=\frac{a_0}{2}+\sum_{n=1}^{\infty}\big[a_n\cos(nx)+b_n\sin(nx)\big],Γ(x)=2a0​​+n=1∑∞​[an​cos(nx)+bn​sin(nx)],

把同頻的 ancos⁡(nx)+bnsin⁡(nx)a_n\cos(nx)+b_n\sin(nx)an​cos(nx)+bn​sin(nx) 用三角恆等式合成單一餘弦：

第 4 步：c0c_0c0​ 的物理意義——DC、控制 1/f1/f1/f upconversion​

把 ISF 的 Fourier 展開代回 LTV phase response（[P1] Eq.(11)→Eq.(13), p.183）：

ϕ(t)=1qmax ⁣[c02 ⁣∫−∞t ⁣in dτ+∑n=1∞cn ⁣∫−∞t ⁣incos⁡(nω0τ+θn) dτ][P1] Eq.(13), p.183\phi(t)=\frac{1}{q_{max}}\!\left[\frac{c_0}{2}\!\int_{-\infty}^{t}\!i_n\,d\tau+\sum_{n=1}^{\infty}c_n\!\int_{-\infty}^{t}\!i_n\cos(n\omega_0\tau+\theta_n)\,d\tau\right]\qquad[\text{P1] Eq.(13), p.183}ϕ(t)=qmax​1​[2c0​​∫−∞t​in​dτ+n=1∑∞​cn​∫−∞t​in​cos(nω0​τ+θn​)dτ][P1] Eq.(13), p.183

第一項（c0c_0c0​ 那項）對 noise 電流直接積分，沒有任何 cos⁡(nω0τ)\cos(n\omega_0\tau)cos(nω0​τ) 調變。意思是：

  • c0c_0c0​ 項對 noise 電流的 DC / 極低頻分量有反應——你慢慢地推節點，相位就慢慢地漂。
  • 把一個靠近 DC 的小單音 i(t)=I0cos⁡(Δωt)i(t)=I_0\cos(\Delta\omega t)i(t)=I0​cos(Δωt)（Δω≪ω0\Delta\omega\ll\omega_0Δω≪ω0​）代入， 只有 c0c_0c0​ 項會響應（其他諧波被 cos⁡(nω0τ)\cos(n\omega_0\tau)cos(nω0​τ) 平均掉），得到（[P1] Eq.(15), p.183）：
ϕ(t)≈I0 c0sin⁡(Δωt)2qmax Δω.\phi(t)\approx\frac{I_0\,c_0\sin(\Delta\omega t)}{2q_{max}\,\Delta\omega}.ϕ(t)≈2qmax​ΔωI0​c0​sin(Δωt)​.
  • 為何這叫 "1/f1/f1/f upconversion（上轉）"：device 的 flicker / 1/f1/f1/f noise 集中在低頻 （DC 附近）。c0c_0c0​ 項把這些「住在 baseband 的 noise」搬（up-convert）到 carrier 旁， 變成 close-in phase noise。後面 flicker_noise_upconversion 會用它推出 1/f31/f^31/f3 skirt（[P1] Eq.(23),(24)）。
  • 設計含義：c0c_0c0​ 越小，flicker 上轉越弱。c0=0c_0=0c0​=0（完美對稱波形）時，理論上 close-in 不再 有 1/f31/f^31/f3 貢獻。這是「波形對稱性 → close-in phase noise」這條 design rule 的數學根源 （[P2] Fig. 17, p.802 有實驗佐證）。

第 5 步：cnc_ncn​ 的物理意義——把 nω0n\omega_0nω0​ 附近的 noise frequency-translate 到 carrier​

看 Eq.(13) 第二項的第 nnn 個分量：它把 noise 電流乘上 cos⁡(nω0τ+θn)\cos(n\omega_0\tau+\theta_n)cos(nω0​τ+θn​) 再積分。 乘上 cos⁡(nω0τ)\cos(n\omega_0\tau)cos(nω0​τ) 在頻域就是把 noise 頻譜搬移 ±nω0\pm n\omega_0±nω0​（調變定理）。所以：

  • 落在 nω0+Δωn\omega_0+\Delta\omeganω0​+Δω（與 nω0−Δωn\omega_0-\Delta\omeganω0​−Δω）附近的 device noise，被 cnc_ncn​ 加權後 down-convert 到 Δω\Delta\omegaΔω，成為 carrier 旁 Δω\Delta\omegaΔω 處的 phase noise。
  • 把一個靠近 nω0n\omega_0nω0​ 的小單音 i(t)=I0cos⁡((nω0+Δω)t)i(t)=I_0\cos((n\omega_0+\Delta\omega)t)i(t)=I0​cos((nω0​+Δω)t) 代入 Eq.(13)， 只有第 nnn 個諧波響應，得到（[P1] Eq.(16)/(17), p.183）：
ϕ(t)≈I0 cnsin⁡(Δωt)2qmax Δω.\phi(t)\approx\frac{I_0\,c_n\sin(\Delta\omega t)}{2q_{max}\,\Delta\omega}.ϕ(t)≈2qmax​ΔωI0​cn​sin(Δωt)​.

第 6 步：為何 close-in（1/f31/f^31/f3）由 c0c_0c0​ 與低階係數主宰​

把上面兩步合起來看一張「頻率地圖」：

第 8 步：如何用數值積分算係數（真實函式）​

理論之外，給一段「會跑」的程式。本站的 compute_fourier_coefficients（Read 自 simulations/common/isf_utils.py）就是把第 2 步的積分用梯形法（trapezoidal rule）做掉：

import numpy as np
from simulations.common.isf_utils import (
    gamma_lc_ideal, gamma_asymmetric,
    compute_fourier_coefficients, reconstruct_from_fourier, gamma_rms,
)

# theta 必須「剛好」涵蓋一個週期 [0, 2*pi]（含端點），梯形法才會給對 (1/pi)*∫ 的值
theta = np.linspace(0.0, 2 * np.pi, 4096, endpoint=True)

# 理想 LC ISF：Gamma(theta) = -sin(theta)（奇函數）
gamma = gamma_lc_ideal(theta)

# a0 即 Hajimiri 記號的 c0；a,b 為 cos/sin 係數；c_n = sqrt(a_n^2+b_n^2)
a0, a, b, c, phase = compute_fourier_coefficients(theta, gamma, n_harmonics=8)

print("c0 =", a0)            # -> ~0.0  (奇函數，無 DC，無 1/f 上轉)
print("c1 =", c[1])         # -> ~1.0  (只有基頻分量)
print("c2..c8 =", c[2:])    # -> ~0    (純單頻)

# 反推回波形，驗證 Eq.(12) 的重建
gamma_hat = reconstruct_from_fourier(theta, a0, a, b)
print("max reconstruction error =", np.max(np.abs(gamma_hat - gamma)))  # -> ~1e-15

I0, c2, qmax, dw = 1e-6, 0.35, 1e-12, 2*np.pi*1e6
phi_p = I0*c2/(2*qmax*dw)
P_sb  = (I0*c2/(4*qmax*dw))**2
print(round(phi_p*1e3,1), "mrad ;", round(10*np.log10(P_sb),1), "dBc")
# -> 27.9 mrad ; -37.1 dBc

重點回顧​

  • Γ\GammaΓ 是 2π2\pi2π 週期函數，可展成 Γ=c02+∑n≥1cncos⁡(nω0τ+θn)\Gamma=\dfrac{c_0}{2}+\sum_{n\ge1}c_n\cos(n\omega_0\tau+\theta_n)Γ=2c0​​+∑n≥1​cn​cos(nω0​τ+θn​)（[P1] Eq.(12), p.183）。
  • 振盪器像時變 mixer：第 nnn 個係數 cnc_ncn​ 把 nω0n\omega_0nω0​ 附近的 device noise down-convert 到 carrier（[P1] Eq.(16), Fig. 8）。
  • c0c_0c0​（DC 係數，DC 值=c0/2=c_0/2=c0​/2）把 baseband 的 flicker noise 上轉成 close-in 1/f31/f^31/f3（[P1] Eq.(15),(23),(24)）。
  • 波形對稱性決定係數的奇偶：奇函數 ⟹ c0=0c_0=0c0​=0（無 1/f1/f1/f 上轉）；半波對稱 ⟹ 偶次諧波歸零。
  • 理想 LC：c0=0, c1=1c_0=0,\ c_1=1c0​=0, c1​=1，其餘為 0；Γrms=1/2≈0.707\Gamma_{rms}=1/\sqrt2\approx0.707Γrms​=1/2​≈0.707。
  • 數值上用 compute_fourier_coefficients（梯形法、theta 含 000 到 2π2\pi2π 端點）即可算係數。
