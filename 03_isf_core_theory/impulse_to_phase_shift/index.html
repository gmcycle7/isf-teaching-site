從 impulse 到 phase shift 的推導

先備：oscillator_phase（limit cycle 與相位／振幅幾何）、lti_vs_ltv（為何振盪器對 noise 是 LTV） ｜ 接下來：isf_definition（嚴謹 ISF 定義與多節點）→ convolution_derivation（對任意 noise 做卷積疊加）

這頁回答一個非常具體的問題：一個 current impulse（電流脈衝）注進振盪器的某個節點， 會讓振盪器的相位偏移多少？ 答案就是 ISF 理論的操作型定義：

Δϕ=Γ(ω0τ)qmax Δq\Delta\phi=\frac{\Gamma(\omega_0\tau)}{q_{max}}\,\Delta qΔϕ=qmax​Γ(ω0​τ)​Δq

我們不直接背這條，而是從電容的 q=Cvq=Cvq=Cv 一步步推出來，每一步都說明用了什麼物理、 什麼近似、單位是什麼、怎麼做 dimension check。

物理直覺（先講結論）：noise 電流先變成節點上的一坨電荷 Δq\Delta qΔq；這坨電荷 把振盪器的瞬時狀態推離原本軌跡一點點；這「一點點」有多少轉成相位、多少轉成振幅， 取決於你在波形的哪個相位踢它——這個「轉換比例」就是 ISF Γ\GammaΓ。在波形斜率大的 地方踢，相位被推得多；在波峰踢，幾乎只改振幅。

第 1 步：current impulse 變成 charge​

電流對時間積分就是電荷。一個很窄、很短的電流脈衝 i(t)i(t)i(t) 注入節點，沉積的總電荷是

Δq=∫i(t) dt.\Delta q=\int i(t)\,dt .Δq=∫i(t)dt.
  • 用到的物理：電荷守恆／電流定義 i=dq/dti=dq/dti=dq/dt。
  • 單位檢查：[A]⋅[s]=[C][\text{A}]\cdot[\text{s}]=[\text{C}][A]⋅[s]=[C] ✓（安培乘秒等於庫侖）。
  • 為何合理：脈衝寬度遠小於週期 TTT 時，可視為「瞬間」把 Δq\Delta qΔq 倒進節點。

第 2 步：charge 變成 voltage 步階​

節點上有總電容 CnodeC_{node}Cnode​。瞬間多出的電荷會讓節點電壓瞬間跳一步（[P1] Eq.(9), p.181）：

ΔV=ΔqCnode.\Delta V=\frac{\Delta q}{C_{node}} .ΔV=Cnode​Δq​.
  • 用到的物理：電容關係 q=Cvq=Cvq=Cv，對小變化取微分 Δq=C ΔV\Delta q=C\,\Delta VΔq=CΔV。
  • 單位檢查：[C]/[F]=[C]/[C/V]=[V][\text{C}]/[\text{F}]=[\text{C}]/[\text{C/V}]=[\text{V}][C]/[F]=[C]/[C/V]=[V] ✓。
  • 關鍵觀察：在並聯 LC 中，電流脈衝只改電容電壓（瞬間），不改電感電流 （電感電流不能瞬變）。所以 state-space 裡這是一個沿電壓軸的水平跳變。

第 3 步：voltage 步階推離 limit cycle，但只有「切向分量」變成相位​

把振盪器狀態畫在 2-D 平面（例如橫軸=電容電壓 vvv、縱軸=正比於電感電流的 www）。穩態時 狀態沿著 limit cycle（極限環）轉圈。第 2 步的電壓跳變把狀態點推到旁邊：

  • 推離環的徑向分量 → 改變振幅 AAA。但振盪器有 amplitude restoring（振幅恢復）機制， 這部分會被慢慢拉回，不會永久殘留。
  • 沿著環的切向分量 → 改變相位 ϕ\phiϕ。相位沒有恢復力，這部分永久留著。

同一個 ΔV\Delta VΔV 在不同相位 τ\tauτ 踢，切向／徑向的分配比例不同。把「ΔV\Delta VΔV 轉成 Δϕ\Delta\phiΔϕ 的比例」整理成一個只跟注入相位有關的週期函數，就是 ISF：

Δϕ=(相位轉換比例)⏟只跟ω0τ有關×ΔV.\Delta\phi=\underbrace{\big(\text{相位轉換比例}\big)}_{\text{只跟}\omega_0\tau\text{有關}}\times\Delta V .Δϕ=只跟ω0​τ有關(相位轉換比例)​​×ΔV.

下圖把這個分解畫在 state plane 上。同一坨電荷造成的水平 ΔV=Δq/C\Delta V=\Delta q/CΔV=Δq/C （電流只踢電容電壓軸，電感電流不能瞬變），被虛線分解成沿環的切向分量（綠，永久相位） 與離環的徑向分量（紅，會衰減）。在 zero-crossing 附近注入時水平踢幾乎全落在切向（相位被推得多）； 在 peak 附近注入時幾乎全落在徑向（只改振幅）。兩格左下角標出 Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ 的值—— 切向佔比恰好就是 Γ(θ)\Gamma(\theta)Γ(θ)，這讓理想 LC 的 Γ=−sin⁡θ\Gamma=-\sin\thetaΓ=−sinθ 變得幾何上不可避免 （Γ=−sin⁡θ\Gamma=-\sin\thetaΓ=−sinθ 的代數推導見 isf_definition）。

  • 用到的數學：把擾動向量投影到 limit cycle 的切線方向（phase direction）。
  • 為何只留切向：見 phase_vs_amplitude_noise 與 oscillator_phase——振幅被 restoring 拉回，相位不會。

第 4 步：用 qmaxq_{max}qmax​ normalize，得到無因次的 ISF​

把第 1～3 步串起來：Δϕ∝ΔV=Δq/Cnode\Delta\phi\propto\Delta V=\Delta q/C_{node}Δϕ∝ΔV=Δq/Cnode​。Hajimiri–Lee 選擇用 節點的最大電荷擺幅 qmax=CnodeVmaxq_{max}=C_{node}V_{max}qmax​=Cnode​Vmax​ 來 normalize，把那個「相位轉換比例」 寫成無因次函數 Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ)（[P1] Eq.(10)–(11), p.182）：

 Δϕ=Γ(ω0τ)qmax Δq \boxed{\ \Delta\phi=\frac{\Gamma(\omega_0\tau)}{q_{max}}\,\Delta q\ } Δϕ=qmax​Γ(ω0​τ)​Δq ​
  • 為什麼要 qmaxq_{max}qmax​ normalization：讓 Γ\GammaΓ 變成無因次、與振幅大小無關的「形狀」， 只描述「波形哪裡敏感」。實際的相位偏移大小由 Δq/qmax\Delta q/q_{max}Δq/qmax​（注入電荷相對訊號電荷的 比例）決定。
  • Γ\GammaΓ 為何無因次：Δϕ\Delta\phiΔϕ 是 rad（無因次），Δq/qmax\Delta q/q_{max}Δq/qmax​ 也是無因次， 所以 Γ\GammaΓ 必須無因次。Dimension check：[rad]=Γ⋅[C]/[C][\text{rad}]=\Gamma\cdot[\text{C}]/[\text{C}][rad]=Γ⋅[C]/[C] ⇒Γ\Rightarrow\Gamma⇒Γ 無因次 ✓。
  • 小訊號假設：第 3 步把投影當成線性，要求 Δq≪qmax\Delta q\ll q_{max}Δq≪qmax​（踢一下不能把振盪器踢翻）。 [P1] Fig. 6 用實際 Colpitts 與 5 級 ring 證實了「Δϕ\Delta\phiΔϕ 與 Δq\Delta qΔq 在小電荷時成正比」。

對應的脈衝響應（為下一章鋪路）​

因為相位步階永久保持，把它寫成脈衝響應就帶一個 unit step u(t−τ)u(t-\tau)u(t−τ)（[P1] Eq.(10)）：

hϕ(t,τ)=Γ(ω0τ)qmax u(t−τ).h_\phi(t,\tau)=\frac{\Gamma(\omega_0\tau)}{q_{max}}\,u(t-\tau).hϕ​(t,τ)=qmax​Γ(ω0​τ)​u(t−τ).

注意它依賴絕對注入時刻 τ\tauτ（透過 Γ(ω0τ)\Gamma(\omega_0\tau)Γ(ω0​τ)），不是只依賴 t−τt-\taut−τ ——這正是 **LTV（線性時變）**的特徵，下一章 convolution_derivation 會用它對任意 noise 電流做疊加。ISF 的完整定義與多節點討論見 isf_definition。

數值例子（建立手感）​

例 A：qmax=1q_{max}=1qmax​=1 pC、Δq=1\Delta q=1Δq=1 fC、Γ=0.5\Gamma=0.5Γ=0.5、f0=5f_0=5f0​=5 GHz。

相位步階：

Δϕ=Γ Δqqmax=0.5×(1×10−15 C)1×10−12 C=5×10−4 rad.\Delta\phi=\frac{\Gamma\,\Delta q}{q_{max}}=\frac{0.5\times(1\times10^{-15}\,\text{C})}{1\times10^{-12}\,\text{C}}=5\times10^{-4}\ \text{rad}.Δϕ=qmax​ΓΔq​=1×10−12C0.5×(1×10−15C)​=5×10−4 rad.

換成度：Δϕ=5×10−4×180π≈0.0286∘\Delta\phi=5\times10^{-4}\times\dfrac{180}{\pi}\approx0.0286^\circΔϕ=5×10−4×π180​≈0.0286∘。

換成 timing error（用 Δt=Δϕ/(2πf0)\Delta t=\Delta\phi/(2\pi f_0)Δt=Δϕ/(2πf0​)）：

Δt=5×10−4 rad2π×5×109 Hz=5×10−43.1416×1010 s≈1.59×10−14 s=15.9 fs.\Delta t=\frac{5\times10^{-4}\ \text{rad}}{2\pi\times5\times10^{9}\ \text{Hz}}=\frac{5\times10^{-4}}{3.1416\times10^{10}}\ \text{s}\approx1.59\times10^{-14}\ \text{s}=15.9\ \text{fs}.Δt=2π×5×109 Hz5×10−4 rad​=3.1416×10105×10−4​ s≈1.59×10−14 s=15.9 fs.
  • Dimension check：[rad]/[rad/s]=[s][\text{rad}]/[\text{rad/s}]=[\text{s}][rad]/[rad/s]=[s] ✓（注意 2πf02\pi f_02πf0​ 的單位是 rad/s）。
  • 手感：在 5 GHz（週期 200 ps）下，一顆 1 fC 的電荷（約 6240 個電子）在最敏感相位 也只造成 ~16 fs 的時間誤差。單顆很小——但 noise 是持續踢的，會積分累積（下一章）。

可用內建函式快速驗證：

from simulations.common.isf_utils import impulse_to_phase_step
from simulations.common.noise_utils import phase_to_time_error

dphi = impulse_to_phase_step(delta_q=1e-15, gamma_value=0.5, qmax=1e-12)
dt   = phase_to_time_error(dphi, f0=5e9)
print(dphi, "rad", dt*1e15, "fs")   # -> 0.0005 rad  15.92 fs

  • noise 電流 → 電荷 Δq\Delta qΔq → 電壓跳變 ΔV=Δq/C\Delta V=\Delta q/CΔV=Δq/C → 經 ISF 投影成相位 Δϕ\Delta\phiΔϕ。
