Lab 02 — 理想 LC 振盪器 toy model：Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ 與電荷線性度

lab_01 給了幾何直覺：相位＝切向、振幅＝徑向。 這個 lab 把它算成一個具體的函數。對一個理想無耗並聯 LC 振盪器，把 current impulse 注進 電容節點，所造成的永久相位偏移可以完全解析地寫出來，得到 ISF：

Γ(θ)=−sin⁡θ,θ=ω0τ.\Gamma(\theta)=-\sin\theta,\qquad \theta=\omega_0\tau.Γ(θ)=−sinθ,θ=ω0​τ.

這是全站第一個「真的有形狀」的 ISF，且它不是 toy 假設而是從 state geometry 推出來的。 我們會 (a) 畫出波形與其 ISF、(b) 用數值模擬驗證 small-signal 下 Δϕ\Delta\phiΔϕ 與 Δq\Delta qΔq 嚴格成 正比、(c) 在 state space 看過零注入如何變成「純相位跳變」。

物理直覺（先講結論）：LC 的狀態 z=(v,w)=A(cos⁡θ,sin⁡θ)z=(v,w)=A(\cos\theta,\sin\theta)z=(v,w)=A(cosθ,sinθ) 在圓上等速轉。 current impulse 只把電容電壓 vvv 水平推一步 Δv=Δq/C\Delta v=\Delta q/CΔv=Δq/C。這一步有多少變成 相位、取決於此刻半徑（指向圓心外）與切向（垂直半徑）的夾角。在波峰（θ=0\theta=0θ=0， v=Av=Av=A）這一步完全沿半徑 → 全是振幅；在過零（θ=π/2\theta=\pi/2θ=π/2，v=0v=0v=0）這一步完全沿切向 → 全是相位。把這個投影寫成相位增量，恰好是 −sin⁡θ-\sin\theta−sinθ。

1. 教學目標​

  • 從物理導出 理想 LC 的 ISF Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ，不靠背、不靠假設。
  • 理解 ISF 的「形狀」如何對應波形：∣Γ∣|\Gamma|∣Γ∣ 最大在過零、為零在波峰。
  • 用數值模擬驗證 small-signal 線性度：Δϕ=Γ Δq/qmax\Delta\phi=\Gamma\,\Delta q/q_{max}Δϕ=ΓΔq/qmax​，即在固定注入相位 下 Δϕ\Delta\phiΔϕ 與 Δq\Delta qΔq 成正比（重現 [P1] Fig. 6 的結論）。
  • 在 state space 看見「過零注入＝沿切向推＝純相位跳變、軌跡半徑不變」。
  • 為 convolution_derivation 與 ISF 的傅立葉 分解（lab_05）準備一個乾淨可解析的範例。

2. 數學模型​

理想無耗並聯 LC：能量在 LLL、CCC 之間無損地來回，狀態在 2-D 平面上等速繞圓。寫成 normalized state（A=1A=1A=1）：

z(θ)=(v, w)=(cos⁡θ,  sin⁡θ),θ=ω0t.z(\theta)=(v,\,w)=(\cos\theta,\;\sin\theta),\qquad \theta=\omega_0 t .z(θ)=(v,w)=(cosθ,sinθ),θ=ω0​t.
  • vvv＝電容電壓（∝cos⁡θ\propto\cos\theta∝cosθ）；www＝（正比於）電感電流（∝sin⁡θ\propto\sin\theta∝sinθ）。
  • 此處 μ=0\mu=0μ=0（無振幅恢復），對應「marginally stable 的理想 LC」。模擬時為了讓軌跡乾淨地 停在環上，我們在 (a)(b) 用很小的 μ=0.3\mu=0.3μ=0.3（弱恢復），在 (c) 用 μ=0\mu=0μ=0 看純旋轉。

impulse → 電壓步階（[P1] Eq.(9), p.182）：

ΔV=ΔqCnode.\Delta V=\frac{\Delta q}{C_{node}}.ΔV=Cnode​Δq​.

它只動 vvv（電感電流不能瞬變），即 state 沿 +v+v+v 軸跳 Δv=Δq/C\Delta v=\Delta q/CΔv=Δq/C。

把電壓步階投影成相位。在 z=(cos⁡θ,sin⁡θ)z=(\cos\theta,\sin\theta)z=(cosθ,sinθ) 處，切向（相位增加方向）的單位向量是 t^=(−sin⁡θ,cos⁡θ)\hat t=(-\sin\theta,\cos\theta)t^=(−sinθ,cosθ)。一個沿 +v+v+v 軸的步階 Δv v^=Δv (1,0)\Delta v\,\hat v=\Delta v\,(1,0)Δvv^=Δv(1,0)，其 切向投影（除以半徑 A=1A=1A=1 換成角度增量）為：

Δϕ=t^⋅(Δv,0)A=(−sin⁡θ) ΔvA=−sin⁡θ  ΔvA.\Delta\phi=\frac{\hat t\cdot(\Delta v,0)}{A}=\frac{(-\sin\theta)\,\Delta v}{A}=-\sin\theta\;\frac{\Delta v}{A}.Δϕ=At^⋅(Δv,0)​=A(−sinθ)Δv​=−sinθAΔv​.

代入 Δv=Δq/C\Delta v=\Delta q/CΔv=Δq/C 並用 qmax=C⋅Aq_{max}=C\cdot Aqmax​=C⋅A：

  Δϕ=−sin⁡θ  Δq/CA=−sin⁡θCA Δq=Γ(θ)qmax Δq,Γ(θ)=−sin⁡θ.  \boxed{\;\Delta\phi=-\sin\theta\;\frac{\Delta q/C}{A}=\frac{-\sin\theta}{C A}\,\Delta q=\frac{\Gamma(\theta)}{q_{max}}\,\Delta q,\qquad \Gamma(\theta)=-\sin\theta.\;}Δϕ=−sinθAΔq/C​=CA−sinθ​Δq=qmax​Γ(θ)​Δq,Γ(θ)=−sinθ.​
  • dimension check：Δϕ\Delta\phiΔϕ 是 rad（無因次）；Δq/qmax=[C]/[C]\Delta q/q_{max}=[\text{C}]/[\text{C}]Δq/qmax​=[C]/[C] 無因次； 故 Γ\GammaΓ 無因次 ✓。Γ\GammaΓ 是 2π2\pi2π 週期，與 [P1] Eq.(10),(11) 的定義一致。
  • 物理檢查：θ=0\theta=0θ=0（波峰）→ Γ=0\Gamma=0Γ=0（只改振幅，與 lab_01 圖二紅點吻合）； θ=π/2\theta=\pi/2θ=π/2（上升過零）→ Γ=−1\Gamma=-1Γ=−1（∣Γ∣|\Gamma|∣Γ∣ 最大，純相位，與綠點吻合）。負號代表 「相位被往後推」的方向慣例。

toy model 聲明：理想無耗 LC + 純 2-D 旋轉 + small-signal 投影，是 pedagogical toy model， 非 transistor-level。真實 LC 有有限 QQQ、harmonic、cyclostationary noise，會讓 Γ\GammaΓ 偏離乾淨的 −sin⁡θ-\sin\theta−sinθ（見 effective_isf）。

4. Python 核心 code​

節錄自 simulations/lab_02_lc_toy_model.py（已對照原始碼）。理想 LC 的 ISF 直接由 gamma_lc_ideal（即 −sin⁡θ-\sin\theta−sinθ）提供；線性度則靠 simulate_lc 在固定過零相位注入不同 Δq/qmax\Delta q/q_{max}Δq/qmax​、再用 excess_phase 量持續相位偏移：

from oscillator_models import simulate_lc, excess_phase
from simulations.common.isf_utils import gamma_lc_ideal

f0, fs = 1.0, 8000.0
theta = np.linspace(0, 2 * np.pi, 400)

# (a) 波形 cos(theta) 與其 ISF -sin(theta)
V     = np.cos(theta)
Gamma = gamma_lc_ideal(theta)          # = -np.sin(theta)

# (b) 線性度：在過零 (theta=pi/2, ISF=-1) 注入一系列 dq/qmax，量持續 Δφ
dq_list = np.linspace(-0.05, 0.05, 11)            # dq / q_max
T = 1.0 / f0
t_inj = 4 * T + (np.pi / 2) / (2 * np.pi * f0)    # 注入時刻對應 theta=pi/2
t_ref, xr, yr = simulate_lc(f0, 10 * T, fs, mu=0.3)
phi_ref = excess_phase(t_ref, xr, yr, f0)

dphi = []
for dq in dq_list:
    t_p, xp, yp = simulate_lc(f0, 10 * T, fs, mu=0.3,
                              impulse_time=t_inj, impulse_dx=dq)
    phi_p = excess_phase(t_p, xp, yp, f0)
    m = t_p >= (10 * T - T)                        # 取最後一週期的穩定值
    dphi.append(np.mean(phi_p[m] - phi_ref[m]))
# theory: Δφ = Γ·Δq/q_max = -1 · (dq/q_max)

excess_phase 把載波旋轉 2πf0t2\pi f_0 t2πf0​t 扣掉，只留擾動造成的相位 （ϕ=unwrap⁡(atan2⁡(y,x))−2πf0t\phi=\operatorname{unwrap}(\operatorname{atan2}(y,x))-2\pi f_0 tϕ=unwrap(atan2(y,x))−2πf0​t），這正是 [P1] Eq.(1) 裡的 ϕ(t)\phi(t)ϕ(t)。

9. 如何解讀圖​

(a) 波形與 ISF：

  • 藍線 V(θ)=cos⁡θV(\theta)=\cos\thetaV(θ)=cosθ（tank 電壓），紅線 Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ（ISF）。
  • 注意兩者相位差 90∘90^\circ90∘：VVV 在波峰時 Γ=0\Gamma=0Γ=0、VVV 過零時 ∣Γ∣=1|\Gamma|=1∣Γ∣=1。物理上完全合理 ——ISF 的大小由波形斜率決定（斜率大 = 切向分量大 = 對相位敏感），而 cos⁡\coscos 的斜率就是 −sin⁡-\sin−sin。這正是「對相位最敏感的時刻是過零、最不敏感的是波峰」的圖像。

(b) 線性度：

  • 紫色圓點是數值模擬量到的持續 Δϕ\Delta\phiΔϕ，黑色虛線是理論 Δϕ=−Δq/qmax\Delta\phi=-\Delta q/q_{max}Δϕ=−Δq/qmax​ （因為注入在 Γ=−1\Gamma=-1Γ=−1 處）。兩者在整個 ±0.05\pm0.05±0.05 範圍內幾乎重合——這就是 small-signal 下 Δϕ\Delta\phiΔϕ 嚴格正比於 Δq\Delta qΔq 的眼見為憑，重現 [P1] Fig. 6。
  • 斜率為 −1-1−1（不是 +1+1+1）證實了 Γ(π/2)=−1\Gamma(\pi/2)=-1Γ(π/2)=−1 的負號。

(c) state space：

  • 黑虛線是未擾動的單位圓；綠線是「在過零點被沿切向踢一下」後的軌跡。關鍵觀察：軌跡 半徑幾乎不變（振幅沒被改），但繞圈的相位被永久挪了一段——這就是「純相位跳變」。 對照若注在波峰，會看到半徑先變大再被拉回、相位幾乎不動（讀者可自行把 t_inj 改到波峰試）。

11. 限制與 approximation​

  • pedagogical toy model，非 transistor-level。Γ=−sin⁡θ\Gamma=-\sin\thetaΓ=−sinθ 是理想無耗 LC 的結果； 真實 LC 有有限 QQQ、波形含諧波、device noise 是 cyclostationary（隨工作點 gating），實際 Γ\GammaΓ 會偏離乾淨的 −sin⁡-\sin−sin，需用 transient/adjoint/Floquet 萃取（見 effective_isf，相關的 PPV/adjoint 不在下載的 5 篇 PDF 內，以標準文獻補充）。
  • small-signal / 線性化：投影成切向假設 Δq≪qmax\Delta q\ll q_{max}Δq≪qmax​。圖 (b) 只掃到 ±0.05 qmax\pm0.05\,q_{max}±0.05qmax​； 更大注入會出現 AM–PM、ISF 本身被改變、甚至把振盪器踢出 limit cycle。
  • 單一節點、單一注入：真實電路多節點、各有自己的 Γ\GammaΓ 與 qmaxq_{max}qmax​；多節點 ISF 討論見 isf_definition。
  • 數值積分誤差：simulate_lc 用固定步長 RK4，弱 μ=0.3\mu=0.3μ=0.3 會讓相位量測有極小偏差； 本 lab 取最後一週期平均以降低瞬態殘留。extract_isf_by_injection 用更小的 Δq/qmax=10−3\Delta q/q_{max}=10^{-3}Δq/qmax​=10−3 掃出整條 Γ(θ)\Gamma(\theta)Γ(θ)，與 −sin⁡θ-\sin\theta−sinθ 最大誤差約 0.001 （見 lab_04）。

重點回顧​

  • 理想 LC 的 ISF 可解析導出：Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ（由「電壓步階沿切向投影」而來）。
  • ∣Γ∣|\Gamma|∣Γ∣ 大小 = 波形斜率：過零最敏感（Γ=−1\Gamma=-1Γ=−1）、波峰不敏感（Γ=0\Gamma=0Γ=0）。
  • small-signal 下 Δϕ=Γ Δq/qmax\Delta\phi=\Gamma\,\Delta q/q_{max}Δϕ=ΓΔq/qmax​，模擬驗證線性且斜率 −1-1−1（[P1] Fig. 6）。
  • 過零注入 = 純相位跳變（state 半徑不變、相位永久挪移）。
  • 來源：[P1] Eq.(9),(10),(11), p.182；Fig. 4, 6, 7(a)。
