Lab 05 — ISF 的傅立葉係數與 Parseval

ISF 是一個 2π2\pi2π 週期函數，所以可以拆成傅立葉級數。為什麼要拆？ 因為相位雜訊 （phase noise）公式裡出現的是 ISF 的傅立葉係數 cnc_ncn​，而不是 Γ\GammaΓ 本身： c0c_0c0​（DC 係數）決定 1/f noise 會不會被上轉（upconvert）成 close-in 相位雜訊；cnc_ncn​ 決定 nω0n\omega_0nω0​ 附近的雜訊被搬到 carrier 的權重；全部係數的平方和經 Parseval（帕塞瓦定理， 時域能量 = 頻域能量）等於 2Γrms22\Gamma_{rms}^22Γrms2​，而 Γrms\Gamma_{rms}Γrms​ 直接決定 1/f² 相位雜訊大小。

這個 lab 做三件事：(1) 算一個 ISF 的 cnc_ncn​ 並用 1、2、4 個諧波重建它；(2) 畫係數頻譜並 數值驗證 Parseval 關係；(3) 把對稱（c0=0c_0=0c0​=0）與不對稱（c0≠0c_0\neq0c0​=0）ISF 並排，說明只有 c0≠0c_0\neq0c0​=0 才會把 1/f noise 上轉。

物理直覺：把 ISF 想成「振盪器這台混頻器的本地振盪（LO）波形」。noise 不是直接進相位的； 它先被 ISF 在頻域「梳子般」採樣——nω0n\omega_0nω0​ 附近的 noise 各自被 cnc_ncn​ 加權、下變頻到 carrier 附近。所以ISF 的形狀（它的諧波內容）才是設計師能動的旋鈕：把波形做對稱 （壓 c0c_0c0​）就能壓 1/f³；把整體 Γrms\Gamma_{rms}Γrms​ 做小就能壓 1/f²。

1. 教學目標​

  • 把 ISF 寫成傅立葉級數 Γ(ω0τ)=c02+∑ncncos⁡(nω0τ+θn)\Gamma(\omega_0\tau)=\dfrac{c_0}{2}+\sum_n c_n\cos(n\omega_0\tau+\theta_n)Γ(ω0​τ)=2c0​​+∑n​cn​cos(nω0​τ+θn​)， 並理解每個係數的物理角色。
  • 用數值積分算出一個給定 ISF 的 cnc_ncn​，並示範「諧波越多、重建越逼近」。
  • 用 Parseval 關係 ∑ncn2=2Γrms2\sum_n c_n^2=2\Gamma_{rms}^2∑n​cn2​=2Γrms2​ 把「頻域係數」與「時域 rms」連起來， 並誠實面對半幅（half-amplitude）慣例下 DC 項的記帳陷阱。
  • 看懂 c0c_0c0​（= ISF 的平均值 × 2）為何是 1/f noise 上轉的「閘門」：對稱波形 c0≈0c_0\approx0c0​≈0 → 1/f³ corner 被推到很低。
  • 對應 [P1] Eq.(12),(20),(24)。

2. 數學模型​

ISF 傅立葉級數（[P1] Eq.(12), p.183）：

Γ(ω0τ)=c02+∑n=1∞cncos⁡(nω0τ+θn)\Gamma(\omega_0\tau)=\frac{c_0}{2}+\sum_{n=1}^{\infty}c_n\cos(n\omega_0\tau+\theta_n)Γ(ω0​τ)=2c0​​+n=1∑∞​cn​cos(nω0​τ+θn​)
  • c0/2c_0/2c0​/2 是 ISF 的平均值（DC 值）。注意 factor of 2：傅立葉係數叫 c0c_0c0​，但 ISF 的 DC值是 c0/2c_0/2c0​/2。這個 factor 在算 1/f³ corner 時極易出錯（見 notation 的符號陷阱）。
  • cn,θnc_n,\theta_ncn​,θn​ 是第 nnn 諧波的幅度與相位；cn=an2+bn2c_n=\sqrt{a_n^2+b_n^2}cn​=an2​+bn2​​， θn=atan2⁡(−bn,an)\theta_n=\operatorname{atan2}(-b_n,a_n)θn​=atan2(−bn​,an​)，其中 an,bna_n,b_nan​,bn​ 是標準 cos/sin 係數。
  • dimension check：Γ\GammaΓ 無因次，cos 無因次，所以每個 cnc_ncn​ 無因次 ✓。

Parseval / rms ISF（[P1] Eq.(20), p.185）：

∑n=0∞cn2=1π∫02π∣Γ(x)∣2dx=2 Γrms2\sum_{n=0}^{\infty}c_n^2=\frac{1}{\pi}\int_0^{2\pi}|\Gamma(x)|^2dx=2\,\Gamma_{rms}^2n=0∑∞​cn2​=π1​∫02π​∣Γ(x)∣2dx=2Γrms2​

它把「所有諧波平方和」與「ISF 的均方 × 2」連起來。物理上：白噪相位雜訊正比於 ∑cn2=2Γrms2\sum c_n^2=2\Gamma_{rms}^2∑cn2​=2Γrms2​（[P1] Eq.(19) → Eq.(21)），所以只要知道 Γrms\Gamma_{rms}Γrms​ 就能算 1/f² 相位雜訊，不必逐一追蹤每個諧波。

半幅慣例的記帳陷阱（本 lab 會親眼看到）：Eq.(20) 左邊的 c0c_0c0​ 必須與 Eq.(12) 的 c0c_0c0​ 用同一個定義。本 lab 的 compute_fourier_coefficients 回傳 c[0] = |a0|，其中 a0=2×(DC 值)a_0=2\times(\text{DC 值})a0​=2×(DC 值)，亦即 c0=a0c_0=a_0c0​=a0​。對一個 DC 值不為零的 ISF，直接把 c[0]**2 丟進 ∑cn2\sum c_n^2∑cn2​ 會把 DC 能量多算一次（因為 cos 項用半幅、DC 項用全幅， 兩者平方權重不同）。這正是下面 Parseval 數值會有 ~10% 落差的原因——我們選擇照實呈現 而非把它藏起來。

1/f³ corner（[P1] Eq.(24), p.185，說明 c0c_0c0​ 為何重要）：

Δω1/f3=ω1/f⋅c022 Γrms2≈ω1/f(c0c1)2\Delta\omega_{1/f^3}=\omega_{1/f}\cdot\frac{c_0^2}{2\,\Gamma_{rms}^2}\approx\omega_{1/f}\left(\frac{c_0}{c_1}\right)^2Δω1/f3​=ω1/f​⋅2Γrms2​c02​​≈ω1/f​(c1​c0​​)2

c0→0c_0\to0c0​→0（對稱波形）⇒\Rightarrow⇒ 1/f³ corner →0\to0→0，close-in 相位雜訊乾淨。

4. Python 核心 code​

傅立葉係數用梯形積分直接算（simulations/common/isf_utils.py，逐字引用）：

def compute_fourier_coefficients(theta, gamma, n_harmonics):
    theta = np.asarray(theta, dtype=float)
    gamma = np.asarray(gamma, dtype=float)

    a0 = (1 / np.pi) * _trapz(gamma, theta)
    a = np.zeros(n_harmonics + 1)
    b = np.zeros(n_harmonics + 1)

    for n in range(1, n_harmonics + 1):
        a[n] = (1 / np.pi) * _trapz(gamma * np.cos(n * theta), theta)
        b[n] = (1 / np.pi) * _trapz(gamma * np.sin(n * theta), theta)

    c = np.sqrt(a ** 2 + b ** 2)
    c[0] = abs(a0)  # c0 magnitude (the DC coefficient)
    phase = np.arctan2(-b, a)

    return a0, a, b, c, phase

rms 與 Parseval 檢查（simulations/lab_05_fourier_isf.py 的 fig_coefficients）：

    a0, a, b, c, ph = compute_fourier_coefficients(theta, g, n_harmonics=8)

    # check Parseval: sum c_n^2 ?= 2 Gamma_rms^2
    parseval_lhs = c[0] ** 2 + np.sum(c[1:] ** 2)

本 lab 用的範例 ISF（刻意做成不對稱，讓多個諧波都非零；+0.25 製造非零 c0c_0c0​）：

def make_isf(theta):
    """A richer asymmetric ISF so several harmonics are non-trivial."""
    return (-np.sin(theta) + 0.35 * np.sin(2 * theta)
            + 0.18 * np.cos(3 * theta) + 0.25)  # the +0.25 sets a non-zero c0

  • gamma_rms 怎麼算：Γrms=12π∫02πΓ2 dθ\Gamma_{rms}=\sqrt{\frac{1}{2\pi}\int_0^{2\pi}\Gamma^2\,d\theta}Γrms​=2π1​∫02π​Γ2dθ​， 與 Eq.(20) 右邊一致。

9. 如何解讀圖​

圖 1（reconstruction）：原始 ISF 含 sin⁡θ,sin⁡2θ,cos⁡3θ\sin\theta,\sin2\theta,\cos3\thetasinθ,sin2θ,cos3θ 三個諧波加 DC。 N=1N=1N=1 只抓主諧波（−sin⁡θ-\sin\theta−sinθ 那條），明顯偏離；N=2N=2N=2 把二次諧波補上、形狀大致對；N=4N=4N=4 已含全部非零諧波，幾乎與原始重合。教學點：ISF 的「形狀」就是它的諧波組合，少數幾個係數 就能描述大部分相位雜訊行為——這正是為什麼相位雜訊公式只需要 c0,c1,…c_0,c_1,\dotsc0​,c1​,… 與 Γrms\Gamma_{rms}Γrms​。

圖 2（coefficients）：c0=0.5c_0=0.5c0​=0.5、c1=1.0c_1=1.0c1​=1.0（主導）、c2=0.35c_2=0.35c2​=0.35、c3=0.18c_3=0.18c3​=0.18，其餘約為零 ——剛好對應 make_isf 的四個項。標題的 Parseval 檢查顯示 ∑cn2=1.405\sum c_n^2=1.405∑cn2​=1.405 對上 2Γrms2=1.2802\Gamma_{rms}^2=1.2802Γrms2​=1.280，不完全相等。這不是 bug，是半幅慣例下 DC 項記帳的後果 （見第 2 節陷阱）：cos 諧波在 ∑\sum∑ 中用半幅平方權重，而 c0=a0=2×c_0=a_0=2\timesc0​=a0​=2×DC 用全幅， 被多算。若把 DC 改成只計 (c0/2)2×2(c_0/2)^2\times2(c0​/2)2×2 的正確權重，兩邊會吻合； 對 DC 值為零的對稱 ISF（如純 −sin⁡θ-\sin\theta−sinθ）則本來就沒有這個落差。 對 1/f²/Γrms2\Gamma_{rms}^2Γrms2​ scaling 與 −20 dB/dec 斜率毫無影響——這只是一個常數記帳慣例， 與 white_noise_to_phase_noise 講的著名 factor-of-2 是同一類事。

圖 3（symmetric vs asymmetric）：綠線對稱於零軸（正負面積相消，c0=0c_0=0c0​=0）；紅線整條被 +0.4+0.4+0.4 抬起，紅色虛線標出它的 DC 值 =c0/2=0.4=c_0/2=0.4=c0​/2=0.4，陰影是被抬起的「直流偏量」。只有這塊 非零 DC 才會把 device 的 1/f flicker noise 直接上轉到 carrier 附近（[P1] Eq.(24)）： c0=0c_0=0c0​=0 → Δω1/f3=0\Delta\omega_{1/f^3}=0Δω1/f3​=0。設計上這就是「把上升/下降邊做對稱、讓波形上下半週對稱」 能壓低 close-in 1/f³ 相位雜訊的數學根據（設計運用見 symmetry）。

重點回顧​

  • ISF 拆成傅立葉級數：Γ=c02+∑cncos⁡(nω0τ+θn)\Gamma=\dfrac{c_0}{2}+\sum c_n\cos(n\omega_0\tau+\theta_n)Γ=2c0​​+∑cn​cos(nω0​τ+θn​)；少數諧波即可重建（圖 1）。
  • 本範例：c0=0.5c_0=0.5c0​=0.5（DC 值 0.25）、c1=1.0c_1=1.0c1​=1.0、c2=0.35c_2=0.35c2​=0.35、c3=0.18c_3=0.18c3​=0.18、Γrms=0.8\Gamma_{rms}=0.8Γrms​=0.8。
  • Parseval ∑cn2=2Γrms2\sum c_n^2=2\Gamma_{rms}^2∑cn2​=2Γrms2​；程式式得 1.4051.4051.405 vs 1.2801.2801.280，落差來自半幅慣例 DC 記帳，不影響 scaling。
  • c0c_0c0​（= 2 × DC 值）是 1/f 上轉的閘門：對稱波形 c0→0c_0\to0c0​→0 → 1/f³ corner → 0（Eq.(24)）。
  • 來源：[P1] Eqs.(12),(20),(24), p.183–185。
