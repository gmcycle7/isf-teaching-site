Lab 08 — 從 L(f) 積分得 rms jitter

這個 lab 把一張 datasheet 上常見的東西——單邊帶相位雜訊曲線 L(f)\mathcal{L}(f)L(f)（dBc/Hz） ——變成設計師真正在意的數字：rms timing jitter（時間抖動均方根） σt\sigma_tσt​。 我們用一個 canonical 情境（f0=5f_0=5f0​=5 GHz、L(1MHz)=−100\mathcal{L}(1\text{MHz})=-100L(1MHz)=−100 dBc/Hz、1/f21/f^21/f2 斜率、 積 1→100 MHz），讓數值積分與解析閉式逐位吻合：σt=447.9\sigma_t=447.9σt​=447.9 fs。

物理直覺（先講結論）：相位雜訊是「每個 offset 頻率上有多少相位功率密度」， jitter 是「全部加起來、開根號、換算成時間」。所以從 L(f)\mathcal{L}(f)L(f) 到 σt\sigma_tσt​ 就是三件事：(1) dBc/Hz 換回 linear 並乘 2 還原 SϕS_\phiSϕ​；(2) 對 offset 頻率積分得相位 variance； (3) 開根號再除以 2πf02\pi f_02πf0​ 換成時間。對 1/f21/f^21/f2 形狀，積分被下限 f1f_1f1​ 主導 ——「從哪裡開始積」比「積到多高」重要得多。

1. 教學目標​

  • 把 L(f)\mathcal{L}(f)L(f)（dBc/Hz）一步步換成 rms phase σϕ\sigma_\phiσϕ​ 與 rms jitter σt\sigma_tσt​。
  • 建立 5 GHz 的換算手感：−100-100−100 dBc/Hz @ 1 MHz、1/f21/f^21/f2、積 1→100 MHz → 約 448 fs。
  • 驗證數值積分 = 解析閉式（1/f21/f^21/f2 有 closed form）。
  • 理解為何 1/f21/f^21/f2 的 jitter 由積分下限主導。

2. 數學模型​

步驟 1：L\mathcal{L}L → SϕS_\phiSϕ​。 小角／單邊帶近似下（規範公式 16）：

L(Δf)≈12Sϕ(Δf)  ⇒  Sϕ(f)=2⋅10L(f)/10.\mathcal{L}(\Delta f)\approx\tfrac12 S_\phi(\Delta f)\;\Rightarrow\;S_\phi(f)=2\cdot10^{\mathcal{L}(f)/10}.L(Δf)≈21​Sϕ​(Δf)⇒Sϕ​(f)=2⋅10L(f)/10.

步驟 2：寫出 1/f21/f^21/f2 形狀（以 fref=1f_{ref}=1fref​=1 MHz 錨定，規範例 C）：

Sϕ(f)=Sϕ(fref)(freff)2.S_\phi(f)=S_\phi(f_{ref})\left(\frac{f_{ref}}{f}\right)^2.Sϕ​(f)=Sϕ​(fref​)(ffref​​)2.

步驟 3：積分得相位 variance（規範公式 18）：

σϕ2=∫f1f2Sϕ(f) df.\sigma_\phi^2=\int_{f_1}^{f_2}S_\phi(f)\,df.σϕ2​=∫f1​f2​​Sϕ​(f)df.

對 1/f21/f^21/f2 形狀，這個積分有閉式（∫f−2df=−1/f\int f^{-2}df=-1/f∫f−2df=−1/f）：

σϕ2=Sϕ(fref) fref2 ⁣∫f1f2dff2=Sϕ(fref) fref2(1f1−1f2).\sigma_\phi^2=S_\phi(f_{ref})\,f_{ref}^2\!\int_{f_1}^{f_2}\frac{df}{f^2}=S_\phi(f_{ref})\,f_{ref}^2\left(\frac{1}{f_1}-\frac{1}{f_2}\right).σϕ2​=Sϕ​(fref​)fref2​∫f1​f2​​f2df​=Sϕ​(fref​)fref2​(f1​1​−f2​1​).

步驟 4：換成 rms jitter（規範公式 19）：

σt=σϕ2πf0=12πf0∫f1f2Sϕ(f) df.\sigma_t=\frac{\sigma_\phi}{2\pi f_0}=\frac{1}{2\pi f_0}\sqrt{\int_{f_1}^{f_2}S_\phi(f)\,df}.σt​=2πf0​σϕ​​=2πf0​1​∫f1​f2​​Sϕ​(f)df​.
  • Dimension check（步驟 4）：σϕ\sigma_\phiσϕ​ 是 rad，2πf02\pi f_02πf0​ 是 rad/s， [rad]/[rad/s]=[s][\text{rad}]/[\text{rad/s}]=[\text{s}][rad]/[rad/s]=[s] ✓。
  • 下限主導：(1f1−1f2)=10−6−10−8=9.9×10−7\big(\tfrac1{f_1}-\tfrac1{f_2}\big)=10^{-6}-10^{-8}=9.9\times10^{-7}(f1​1​−f2​1​)=10−6−10−8=9.9×10−7， 其中 1/f1=10−61/f_1=10^{-6}1/f1​=10−6 佔 99%。所以積分結果幾乎只看 f1f_1f1​。

逐位代入 canonical 數字（與 numerical_feeling 例 C 完全一致）：

Sϕ(1MHz)=2×10−100/10=2×10−10 rad2/Hz,σϕ2=2×10−10 (106)2 (10−6−10−8)=200×9.9×10−7=1.98×10−4 rad2,σϕ=1.407×10−2 rad=14.07 mrad,σt=1.407×10−22π×5×109=4.479×10−13 s=447.9 fs.\begin{aligned} S_\phi(1\text{MHz})&=2\times10^{-100/10}=2\times10^{-10}\ \text{rad}^2/\text{Hz},\\ \sigma_\phi^2&=2\times10^{-10}\,(10^6)^2\,(10^{-6}-10^{-8})=200\times9.9\times10^{-7}=1.98\times10^{-4}\ \text{rad}^2,\\ \sigma_\phi&=1.407\times10^{-2}\ \text{rad}=14.07\ \text{mrad},\\ \sigma_t&=\frac{1.407\times10^{-2}}{2\pi\times5\times10^{9}}=4.479\times10^{-13}\ \text{s}=447.9\ \text{fs}. \end{aligned}Sϕ​(1MHz)σϕ2​σϕ​σt​​=2×10−100/10=2×10−10 rad2/Hz,=2×10−10(106)2(10−6−10−8)=200×9.9×10−7=1.98×10−4 rad2,=1.407×10−2 rad=14.07 mrad,=2π×5×1091.407×10−2​=4.479×10−13 s=447.9 fs.​

4. Python 核心 code​

逐字摘自 simulations/lab_08_jitter_integration.py 的 main()：先用 leeson_one_over_f2 造出 1/f21/f^21/f2 曲線，再以 integrate_rms_jitter 做數值積分， 最後手算 1/f21/f^21/f2 閉式對拍。

import numpy as np
from simulations.common.noise_utils import leeson_one_over_f2, integrate_rms_jitter

f0 = 5e9
f_ref = 1e6
L_ref = -100.0  # dBc/Hz
f1, f2 = 1e6, 100e6

f = np.logspace(np.log10(f1), np.log10(f2), 4000)
L = leeson_one_over_f2(f, L_ref, f_ref)

# numerical integration
sigma_t, sigma_phi = integrate_rms_jitter(f, L, f0, f1, f2)

# analytic closed form (1/f^2)
L_ref_lin = 10 ** (L_ref / 10)
sigma_phi2_analytic = 2 * L_ref_lin * f_ref ** 2 * (1 / f1 - 1 / f2)
sigma_phi_analytic = np.sqrt(sigma_phi2_analytic)
sigma_t_analytic = sigma_phi_analytic / (2 * np.pi * f0)

print(round(sigma_t * 1e15, 1))    # -> 447.9   (sigma_t in fs)
print(round(sigma_phi * 1e3, 2))   # -> 14.07   (sigma_phi in mrad)

底層函式 integrate_rms_jitter（noise_utils.py）做的就是步驟 1、3、4：

l_linear   = 10 ** (l_dbc_per_hz[mask] / 10)   # dBc/Hz -> linear
s_phi      = 2 * l_linear                       # L ~= 0.5 * S_phi
sigma_phi  = np.sqrt(_trapz(s_phi, f[mask]))    # 積分 + 開根號
sigma_t    = sigma_phi / (2 * np.pi * f0)        # 換成時間
  • 程式印出 sigma_phi (numeric) ≈ sigma_phi (analytic)、sigma_t ≈ 447.9 fs， 數值與解析逐位一致（差異只來自 logspace 取樣與梯形積分的離散誤差，可忽略）。

9. 如何解讀圖​

  • 藍線（L(f)L(f)L(f)）：在 semilog（x 軸對數）上是一條直線，因為 1/f21/f^21/f2 在 dBc/Hz 對 log-f 下斜率固定（−20-20−20 dB/dec）。紅點標出錨定點 L(1MHz)=−100\mathcal{L}(1\text{MHz})=-100L(1MHz)=−100 dBc/Hz。
  • 藍色陰影：示意「相位功率分布在哪個 offset 頻率」。1/f21/f^21/f2 下，最靠近 f1f_1f1​ 的部分 貢獻最多——這正是「下限主導」的視覺呈現。
  • 文字框：列出積分結果 σϕ=14.07\sigma_\phi=14.07σϕ​=14.07 mrad、σt=447.9\sigma_t=447.9σt​=447.9 fs，並標註解析值 與數值值相同，證明 closed form 與梯形積分一致。
  • 怎麼用：拿到任何 datasheet 的 L(f)\mathcal{L}(f)L(f) 曲線，照藍線下的面積積分、開根號、 除以 2πf02\pi f_02πf0​，就得到 jitter。記住 5 GHz 換算點：本例 ≈448\approx448≈448 fs。 若相位雜訊好 20 dB（−120-120−120 dBc/Hz @ 1 MHz），功率小 100 倍、σ\sigmaσ 小 10 倍 → ≈45\approx45≈45 fs。

10. 對應 paper 公式/figure​

  • L≈12Sϕ\mathcal{L}\approx\frac12 S_\phiL≈21​Sϕ​：規範公式 16（小角近似）。

  • rms jitter：規範公式 19，σt=12πf0∫Sϕdf\sigma_t=\frac{1}{2\pi f_0}\sqrt{\int S_\phi df}σt​=2πf0​1​∫Sϕ​df​。

11. 限制與 approximation​

  • 小角近似 L≈12Sϕ\mathcal{L}\approx\frac12 S_\phiL≈21​Sϕ​：相位偏移大時失準；本例 σϕ=14\sigma_\phi=14σϕ​=14 mrad ≪1\ll1≪1 rad， 近似很好。
  • 純 1/f21/f^21/f2 skirt 假設：真實曲線還有 1/f31/f^31/f3（close-in，見 lab_07） 與遠端的 white floor（雜訊地板）。本 lab 只取單一 1/f21/f^21/f2 段，是教學簡化。 這部分非 toy（積分流程本身是真實工程做法），但 L(f)L(f)L(f) 形狀是理想化的。
  • 積分範圍敏感：1/f21/f^21/f2 由下限 f1f_1f1​ 主導，換 f1f_1f1​ 會明顯改 σt\sigma_tσt​；上限影響小。 實務上 f1f_1f1​、f2f_2f2​ 要依應用（PLL 環路頻寬、資料率）選定。
  • 數值=解析：兩者只差離散積分誤差（logspace + 梯形法），在本設定下可忽略。

重點回顧​

  • L\mathcal{L}L(dBc/Hz) → linear → ×2\times2×2 得 SϕS_\phiSϕ​ → 積分 → 開根號 → ÷(2πf0)\div(2\pi f_0)÷(2πf0​) = rms jitter。
  • canonical：5 GHz、−100-100−100 dBc/Hz @ 1 MHz、1/f21/f^21/f2、積 1→100 MHz → σϕ=14.07\sigma_\phi=14.07σϕ​=14.07 mrad、σt=447.9\sigma_t=447.9σt​=447.9 fs。
  • 1/f21/f^21/f2 的 jitter 由積分下限主導（1/f11/f_11/f1​ 項）。
  • 數值積分與 1/f21/f^21/f2 解析閉式逐位一致。
