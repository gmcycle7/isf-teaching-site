Lab 09 — 設計取捨綜合 scaling

麵包屑：模擬實驗室 › 雜訊與抖動 › 本頁（設計取捨入門）。上游：lab_06、lab_08；進階版：lab_17。

前面八個 lab 把每個機制各別跑通了。這個綜合 lab 不引入新模擬，而是把 [P1] Eq.(21) 這條招牌式當成設計旋鈕地圖：固定其他條件，分別掃 qmaxq_{max}qmax​、Γrms\Gamma_{rms}Γrms​、 ring 級數 NNN，看 phase noise L\mathcal{L}L 與 rms jitter σt\sigma_tσt​ 怎麼動。 全程用 canonical 數值（規範第 8 節）做口算，建立「動哪個旋鈕、賺幾 dB」的反射。

物理直覺（先講結論）：1/f21/f^21/f2 phase noise ∝Γrms2/qmax2\propto\Gamma_{rms}^2/q_{max}^2∝Γrms2​/qmax2​。 想降相位雜訊只有兩條大路：把訊號電荷 qmaxq_{max}qmax​ 做大（多存能量，雜訊相對變小）， 或把 ISF Γrms\Gamma_{rms}Γrms​ 做小（讓波形對雜訊不敏感）。 兩者都是 20log⁡1020\log_{10}20log10​ 的關係：qmaxq_{max}qmax​ 加倍 → 相位雜訊 −6-6−6 dB； Γrms\Gamma_{rms}Γrms​ 減半 → 也是 −6-6−6 dB。ring 振盪器的 NNN 同時動兩件事，要小心淨效果。

1. 教學目標​

  • 把 [P1] Eq.(21) 讀成 scaling law，建立 qmaxq_{max}qmax​、Γrms\Gamma_{rms}Γrms​、NNN 的設計直覺。
  • 用 canonical 數值算出「動一個旋鈕賺幾 dB / 幾 fs」。
  • 看懂 ring 振盪器 NNN 的雙面性（Γrms∝N−3/4\Gamma_{rms}\propto N^{-3/4}Γrms​∝N−3/4，但功率/面積也漲）。
  • 把 scaling 結論連到各 design insight 頁。

2. 數學模型​

起點是 [P1] Eq.(21), p.185：

L{Δω}=10log⁡10 ⁣(Γrms2qmax2⋅in2‾/Δf4 Δω2)\mathcal{L}\{\Delta\omega\}=10\log_{10}\!\left(\frac{\Gamma_{rms}^2}{q_{max}^2}\cdot\frac{\overline{i_n^2}/\Delta f}{4\,\Delta\omega^2}\right)L{Δω}=10log10​(qmax2​Γrms2​​⋅4Δω2in2​​/Δf​)

把對數展開成「相加的旋鈕」（固定 Δω\Delta\omegaΔω、Si=in2‾/ΔfS_i=\overline{i_n^2}/\Delta fSi​=in2​​/Δf）：

L=20log⁡10Γrms−20log⁡10qmax+10log⁡10Si−20log⁡10(2Δω)+const.\mathcal{L}=20\log_{10}\Gamma_{rms}-20\log_{10}q_{max}+10\log_{10}S_i-20\log_{10}(2\Delta\omega)+\text{const}.L=20log10​Γrms​−20log10​qmax​+10log10​Si​−20log10​(2Δω)+const.
  • 讀法：每個旋鈕都是 20log⁡1020\log_{10}20log10​（電壓/電荷類）或 10log⁡1010\log_{10}10log10​（功率類）。 qmaxq_{max}qmax​ 與 Γrms\Gamma_{rms}Γrms​ 都進 20log⁡1020\log_{10}20log10​，所以「加倍/減半 = ±6\pm6±6 dB」。

jitter 與 phase noise 同源：對 1/f21/f^21/f2 skirt，σt=12πf0∫Sϕdf\sigma_t=\frac{1}{2\pi f_0}\sqrt{\int S_\phi df}σt​=2πf0​1​∫Sϕ​df​ （規範公式 19），而 Sϕ∝Γrms2/qmax2S_\phi\propto\Gamma_{rms}^2/q_{max}^2Sϕ​∝Γrms2​/qmax2​，所以

σt  ∝  Γrmsqmax(固定f0, Si, 積分範圍).\sigma_t\;\propto\;\frac{\Gamma_{rms}}{q_{max}}\qquad(\text{固定}f_0,\ S_i,\ \text{積分範圍}).σt​∝qmax​Γrms​​(固定f0​, Si​, 積分範圍).
  • dB 與 jitter 的橋：L\mathcal{L}L 降 666 dB（功率 ×14\times\frac14×41​）→ σt\sigma_tσt​ 減半（\sqrt{}​）。

ring 振盪器：[P2] 給出兩條 scaling——級數越多，每級 ISF 越尖但 rms 越小（[P2] Eq.(16), p.794）：

Γrms∝N−3/4([P2] Eq.(16), p.794，已核實),f0=12NτD [P2] Eq.(15).\Gamma_{rms}\propto N^{-3/4}\quad(\text{[P2] Eq.(16), p.794，已核實}),\qquad f_0=\frac{1}{2N\tau_D}\ \text{[P2] Eq.(15)}.Γrms​∝N−3/4([P2] Eq.(16), p.794，已核實),f0​=2NτD​1​ [P2] Eq.(15).
  • 單看 phase noise，NNN 從 5 → 15（×3\times3×3）：Γrms\Gamma_{rms}Γrms​ 乘 3−3/2=0.1923^{-3/2}=0.1923−3/2=0.192， L\mathcal{L}L 變化 20log⁡100.192=−14.320\log_{10}0.192=-14.320log10​0.192=−14.3 dB。但級數多 → 功率、面積、qmaxq_{max}qmax​ 分配都變， 且 f0f_0f0​ 下降（除非縮 τD\tau_DτD​），所以這只是「孤立看 Γrms\Gamma_{rms}Γrms​」的 toy scaling，非淨設計結論。

4. Python 核心 code​

本 lab 不產生新圖，scaling 直接用 [P1] Eq.(21) 口算。下面這段（沿用 simulations/lab_08_jitter_integration.py 的習慣）示範如何把旋鈕變化換成 dB 與 fs， 讓你能自己掃參數：

import math

def L_dbc(Grms, qmax, Si, df):
    """Hajimiri-Lee Eq.(21): 1/f^2 SSB phase noise [dBc/Hz]."""
    dw = 2 * math.pi * df
    return 10 * math.log10((Grms**2 / qmax**2) * (Si / (4 * dw**2)))

# canonical baseline (規範例 B)
f0, df, Si = 5e9, 1e6, 1e-24
base = L_dbc(Grms=0.5, qmax=1e-12, Si=Si, df=df)   # -148.0 dBc/Hz

print("baseline           :", round(base, 1), "dBc/Hz")
print("q_max x2           :", round(L_dbc(0.5, 2e-12, Si, df) - base, 1), "dB")  # -6.0
print("Gamma_rms /2       :", round(L_dbc(0.25, 1e-12, Si, df) - base, 1), "dB") # -6.0
# ring N: Gamma_rms ~ N^-3/4 ([P2] Eq.(16), p.794, verified); show isolated scaling
for N in (5, 15, 45):
    rel = (N / 5.0) ** -1.5
    print(f"N={N:2d} Gamma_rms rel={rel:.3f}  dL={20*math.log10(rel):+.1f} dB")

# jitter scales as Gamma_rms / q_max  ->  -6 dB phase noise = sigma_t / 2
  • L_dbc 與 lab_08 的 integrate_rms_jitter 不同：lab_08 從 L\mathcal{L}L 積分出 jitter； 這裡反向用 Eq.(21) 產生 L\mathcal{L}L，再用 scaling 推 jitter。兩頁互為正反操作。

11. 限制與 approximation​

  • toy scaling，非 transistor-level：scaling 假設「只動一個旋鈕、其他完全不變」。 真實電路裡 qmaxq_{max}qmax​、Γrms\Gamma_{rms}Γrms​、NNN、功率、面積、f0f_0f0​ 彼此耦合，不能孤立調。
  • ring NNN scaling 已核實：Γrms∝N−3/4\Gamma_{rms}\propto N^{-3/4}Γrms​∝N−3/4 的比例常數為 [P2] Eq.(16), p.794 Γrms=2π2/(3η3)⋅N−3/4\Gamma_{rms}=\sqrt{2\pi^2/(3\eta^3)}\cdot N^{-3/4}Γrms​=2π2/(3η3)​⋅N−3/4（η≈1\eta\approx1η≈1，已核實）；級數變多也改 f0=1/(2NτD)f_0=1/(2N\tau_D)f0​=1/(2NτD​)、功率與面積，淨 phase noise/jitter 要看完整 FOM，不能只看 Γrms\Gamma_{rms}Γrms​。
  • 單一白噪源：忽略多源、cyclostationary（Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α）、flicker 上轉 （1/f31/f^31/f3，見 lab_07）。close-in 區此表不適用。
  • factor-of-2：用 Eq.(21) 的分母 4Δω24\Delta\omega^24Δω2；與時域版差 SSB 記帳 2 倍， 不影響任何 scaling 結論（所有 dB 變化都是相對量）。
  • jitter scaling：σt∝Γrms/qmax\sigma_t\propto\Gamma_{rms}/q_{max}σt​∝Γrms​/qmax​ 假設固定積分範圍與 1/f21/f^21/f2 形狀； 若 close-in 1/f31/f^31/f3 主導，jitter 由 c0c_0c0​ 與積分下限決定，需另算。

重點回顧​

  • 1/f21/f^21/f2 phase noise ∝Γrms2/qmax2\propto\Gamma_{rms}^2/q_{max}^2∝Γrms2​/qmax2​；jitter ∝Γrms/qmax\propto\Gamma_{rms}/q_{max}∝Γrms​/qmax​。
  • qmaxq_{max}qmax​ 加倍或 Γrms\Gamma_{rms}Γrms​ 減半 → phase noise −6-6−6 dB、jitter 減半（20log⁡1020\log_{10}20log10​）。
  • ring NNN 孤立看 Γrms∝N−3/4\Gamma_{rms}\propto N^{-3/4}Γrms​∝N−3/4（N×3→−14.3N\times3\to-14.3N×3→−14.3 dB），但 f0f_0f0​/功率/面積同時變，是 toy scaling。
  • 基準數字：qmax=1q_{max}=1qmax​=1 pC、Γrms=0.5\Gamma_{rms}=0.5Γrms​=0.5、5 GHz @ 1 MHz、Si=10−24S_i=10^{-24}Si​=10−24 → L=−148.0\mathcal{L}=-148.0L=−148.0 dBc/Hz。
