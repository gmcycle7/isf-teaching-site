Lab 14 — Cyclostationary noise 與 effective ISF

麵包屑：模擬實驗室 › 系統與進階 › 本頁（cyclostationary／effective ISF）。上游：lab_06、effective_isf；相關：lab_09。

前面幾個 lab 都假設 noise 源「整個週期都一樣吵」（stationary，定常）。真實電路不是這樣： 一顆 tail current（尾電流源）或 switching transistor（切換電晶體）只在週期的某一段才導通、 才注入 noise。這種「noise 強度本身隨振盪相位週期性變化」的 noise 叫 cyclostationary noise（週期穩態雜訊）。這個 lab 用一個介於 0、1 之間的 noise-modulating function（NMF，雜訊調變函數） α(θ)\alpha(\theta)α(θ) 把「device 何時吵」 畫出來，然後說明真正決定 phase noise 的不是原始 ISF，而是 effective ISF（有效 ISF） Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α。

物理直覺（先講結論）：phase noise 不只看 device 多吵（in2‾\overline{i_n^2}in2​​），也看 它在波形的哪個相位吵。同一顆 device，若它最吵的時段剛好落在 ISF 很大的地方 （例如 zero crossing），它的破壞力就大；若落在 ISF≈0 的波峰，破壞力就小。把 「何時吵」α(θ)\alpha(\theta)α(θ) 乘進 ISF，得到 Γeff=Γα\Gamma_{eff}=\Gamma\alphaΓeff​=Γα，再對 Γeff\Gamma_{eff}Γeff​ 取 rms，才是這顆 cyclostationary 源真正的有效敏感度 Γeff,rms\Gamma_{eff,rms}Γeff,rms​。

1. 教學目標​

  • 理解 stationary（定常）與 cyclostationary（週期穩態）noise 的差別。
  • 用 NMF α(θ)∈[0,1]\alpha(\theta)\in[0,1]α(θ)∈[0,1] 模型化「device 只在週期的某段吵」。
  • 推出 effective ISF Γeff(θ)=Γ(θ) α(θ)\Gamma_{eff}(\theta)=\Gamma(\theta)\,\alpha(\theta)Γeff​(θ)=Γ(θ)α(θ)，並用它取代 Γ\GammaΓ 算 rms。
  • 看懂：同一顆 device、同樣 noise 量，只因「注入相位」不同，Γeff,rms\Gamma_{eff,rms}Γeff,rms​ 可以差好幾倍。

2. 數學模型​

從 [P1] Eq.(21), p.185 的白噪 phase-noise 律出發：

L{Δω}=10log⁡10 ⁣(Γrms2qmax2⋅in2‾/Δf4 Δω2)\mathcal{L}\{\Delta\omega\}=10\log_{10}\!\left(\frac{\Gamma_{rms}^2}{q_{max}^2}\cdot\frac{\overline{i_n^2}/\Delta f}{4\,\Delta\omega^2}\right)L{Δω}=10log10​(qmax2​Γrms2​​⋅4Δω2in2​​/Δf​)

這條假設 in2‾/Δf\overline{i_n^2}/\Delta fin2​​/Δf 是常數（stationary）。若 noise 是 cyclostationary， 其瞬時 PSD 可寫成「定常分量乘上一個週期 gate」：

in2(θ)‾/Δf=(in2‾/Δf)⋅α2(θ),α(θ)∈[0,1].\overline{i_n^2(\theta)}/\Delta f=\big(\overline{i_n^2}/\Delta f\big)\cdot\alpha^2(\theta),\qquad \alpha(\theta)\in[0,1].in2​(θ)​/Δf=(in2​​/Δf)⋅α2(θ),α(θ)∈[0,1].

把這個相位相依塞回 [P1] Eq.(11), p.182 的 ISF 積分核 Γ(ω0τ) in(τ)\Gamma(\omega_0\tau)\,i_n(\tau)Γ(ω0​τ)in​(τ)， 等效於把 noise 的「α\alphaα」併進 ISF。定義 effective ISF（規範第 10 節）：

Γeff(θ)=Γ(θ) α(θ).\Gamma_{eff}(\theta)=\Gamma(\theta)\,\alpha(\theta).Γeff​(θ)=Γ(θ)α(θ).

於是 cyclostationary 源的 phase noise 就用 Γeff,rms\Gamma_{eff,rms}Γeff,rms​ 取代 Γrms\Gamma_{rms}Γrms​：

Γeff,rms=12π∫02π[Γ(θ) α(θ)]2 dθ,\Gamma_{eff,rms}=\sqrt{\frac{1}{2\pi}\int_0^{2\pi}\big[\Gamma(\theta)\,\alpha(\theta)\big]^2\,d\theta},Γeff,rms​=2π1​∫02π​[Γ(θ)α(θ)]2dθ​,

並用它代入 [P1] Eq.(21)：L∝Γeff,rms2/qmax2\mathcal{L}\propto\Gamma_{eff,rms}^2/q_{max}^2L∝Γeff,rms2​/qmax2​。

  • 為何乘法是對的：α\alphaα 描述「noise 振幅相對被 gate 的比例」，ISF 描述「該相位的 noise 轉成相位的效率」。兩者都作用在「同一時刻的注入」上，效果相乘。
  • Dimension check：Γ\GammaΓ 無因次、α\alphaα 無因次（∈[0,1]\in[0,1]∈[0,1]）⇒ Γeff\Gamma_{eff}Γeff​ 無因次 ✓， 和原 ISF 同單位，可直接代進 Eq.(21)。

本 lab 用理想 LC 的 Γ(θ)=−sin⁡θ\Gamma(\theta)=-\sin\thetaΓ(θ)=−sinθ（[P1] 推導，最大 ∣Γ∣\vert\Gamma\vert∣Γ∣ 在 zero crossing θ=π/2\theta=\pi/2θ=π/2、Γ≈0\Gamma\approx0Γ≈0 在波峰 θ=0\theta=0θ=0），配兩個寬度同為 π\piπ 的 平滑 gate：一個 gate 在 zero crossing（壞），一個在波峰（好）。

4. Python 核心 code​

下面是 simulations/lab_14_cyclostationary_isf.py 的核心：用 gamma_lc_ideal（−sin⁡-\sin−sin） 當 ISF，用平滑週期窗 nmf_window 當 α(θ)\alpha(\theta)α(θ)，再用 effective_isf（即 Γ⋅α\Gamma\cdot\alphaΓ⋅α） 與 gamma_rms 算各自的 rms。

import numpy as np
from simulations.common.isf_utils import gamma_lc_ideal, gamma_rms, effective_isf


def nmf_window(theta, center, width):
    """Smooth periodic gate in [0,1], centered at `center`, of given width (rad)."""
    d = np.angle(np.exp(1j * (theta - center)))  # wrapped distance
    return 0.5 * (1 + np.cos(np.pi * np.clip(d / (width / 2), -1, 1)))


theta = np.linspace(0, 2 * np.pi, 2000, endpoint=True)
gamma = gamma_lc_ideal(theta)                 # -sin(theta)

# case 1: device noisy near the ZERO CROSSING (where |Gamma| is max) -> bad
a_bad = nmf_window(theta, center=np.pi / 2, width=np.pi)
# case 2: device noisy near the PEAK (where |Gamma|~0) -> good
a_good = nmf_window(theta, center=0.0, width=np.pi)

g_bad = effective_isf(gamma, a_bad)           # Gamma * alpha
g_good = effective_isf(gamma, a_good)

grms = gamma_rms(theta, gamma)                # stationary reference
grms_bad = gamma_rms(theta, g_bad)
grms_good = gamma_rms(theta, g_good)
print(f"stationary={grms:.3f} bad={grms_bad:.3f} good={grms_good:.3f}")
# -> stationary=0.707, bad=0.395, good=0.177

effective_isf(gamma_values, alpha_values) 就是逐點相乘（見 simulations/common/isf_utils.py）， gamma_rms(theta, gamma) 是 12π∫02πΓ2 dθ\sqrt{\frac{1}{2\pi}\int_0^{2\pi}\Gamma^2\,d\theta}2π1​∫02π​Γ2dθ​，對應 [P1] Eq.(20)。

9. 如何解讀圖​

  • 左圖 (a)：黑線是 ISF Γ=−sin⁡θ\Gamma=-\sin\thetaΓ=−sinθ（在 θ/2π=0.25\theta/2\pi=0.25θ/2π=0.25、0.750.750.75 即 zero crossing 處 ∣Γ∣\vert\Gamma\vert∣Γ∣ 最大；在 000、0.50.50.5 即波峰處 Γ=0\Gamma=0Γ=0）。紅虛線 α\alphaα 把 device 的吵 時段放在 zero crossing（θ/2π≈0.25\theta/2\pi\approx0.25θ/2π≈0.25）；綠虛線 α\alphaα 放在波峰（θ/2π≈0\theta/2\pi\approx0θ/2π≈0）。 兩個 gate 形狀、寬度完全一樣，只差相位。
  • 右圖 (b)：把左圖相乘得 Γeff=Γα\Gamma_{eff}=\Gamma\alphaΓeff​=Γα。紅線（壞）在 zero crossing 撞上 ISF 的 大值，留下一個又深又寬的 lobe，Γeff,rms=0.395\Gamma_{eff,rms}=0.395Γeff,rms​=0.395；綠線（好）落在波峰，ISF 幾乎為 0， 只剩兩側殘餘小 lobe，Γeff,rms=0.177\Gamma_{eff,rms}=0.177Γeff,rms​=0.177。
  • 關鍵讀法：stationary 參考 Γrms=0.707\Gamma_{rms}=0.707Γrms​=0.707（=1/21/\sqrt21/2​，−sin⁡-\sin−sin 整週的 rms）。 一旦 gate 只開半週，rms 一定下降；但下降多少完全看 gate 對到 ISF 的哪裡：壞 case 仍有 0.3950.3950.395，好 case 砍到 0.1770.1770.177，差約 2.22.22.2 倍。 phase noise 差 20log⁡10(0.395/0.177)≈720\log_{10}(0.395/0.177)\approx720log10​(0.395/0.177)≈7 dB。同一顆 device、同樣 noise 量，只改注入相位就賺 7 dB。

11. 限制與 approximation​

  • toy model，非 transistor-level：α(θ)\alpha(\theta)α(θ) 用人工 raised-cosine 窗，不是從電晶體 gmg_mgm​／導通區間萃取的真實 NMF；ISF 用理想 LC 的 −sin⁡-\sin−sin。真實 device 的 α\alphaα 與 Γ\GammaΓ 都要靠 PSS／transient 模擬萃取。
  • Γeff=Γα\Gamma_{eff}=\Gamma\alphaΓeff​=Γα 是一階近似：假設 noise 振幅被 gate 線性調變、且 α\alphaα 與相位 響應相互獨立；忽略 noise 自身的 AM–PM 與 α\alphaα、Γ\GammaΓ 間的耦合。
  • 只示意 rms，未跑 PSD：本 lab 用 Γeff,rms\Gamma_{eff,rms}Γeff,rms​ 說明 scaling，沒有把 cyclostationary 源實際丟進 [P1] Eq.(11) 積分跑出 PSD（那會是 lab_06 的延伸）。
  • 單一源：真實電路多個 cyclostationary 源各有自己的 α\alphaα，要分別算 Γeff,rms\Gamma_{eff,rms}Γeff,rms​ 再對功率求和。
  • factor-of-2：沿用 Eq.(21) 的 4Δω24\Delta\omega^24Δω2 慣例；SSB 記帳的 2 倍差異不影響此處所有 相對（dB）比較。

重點回顧​

  • cyclostationary noise：noise 強度隨振盪相位週期變化，用 α(θ)∈[0,1]\alpha(\theta)\in[0,1]α(θ)∈[0,1] 描述。
  • effective ISF：Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α；phase noise 用 Γeff,rms\Gamma_{eff,rms}Γeff,rms​ 取代 Γrms\Gamma_{rms}Γrms​ 代入 [P1] Eq.(21)。
  • 本 lab 數字：stationary Γrms=0.707\Gamma_{rms}=0.707Γrms​=0.707；noise 在 zero crossing → 0.3950.3950.395（壞）；在波峰 → 0.1770.1770.177（好）。
  • 設計含義：把吵的 device 安排在 ISF 小的相位（波峰）注入，可省可觀 phase noise。
