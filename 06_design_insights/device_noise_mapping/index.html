Device noise → ISF harmonics 的映射

先備：white_noise_to_phase_noise（white → 1/f²）、flicker_noise_upconversion（flicker → 1/f³）、fourier_series_of_isf（c0c_0c0​、cnc_ncn​）｜ 接下來：symmetry、tank_swing、serdes_clocking_connection

這頁把前面零散的設計直覺收成一張地圖：一顆 transistor 的雜訊（white 與 flicker） 是怎麼一路變成 oscillator 的 phase noise 的？關鍵是兩件事——哪一段頻率的 device noise 被哪一個 ISF 諧波接收，以及 device 不是整個週期都在漏雜訊（cyclostationary，週期穩態）， 所以真正起作用的是 effective ISF Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α。

物理直覺（先講結論）：把 ISF 想成一台多頻道收音機。它的傅立葉係數 cnc_ncn​ 是各頻道 的天線增益：c0c_0c0​ 收「DC 附近」的 device flicker（上轉成 1/f³）；c1c_1c1​ 收「ω0\omega_0ω0​ 附近」、 c2c_2c2​ 收「2ω02\omega_02ω0​ 附近」……的 device white noise（下轉成 1/f²）。而 device 只在波形某些相位 才真的在導通、才真的在漏雜訊——這個「何時漏」的開關函數 α(ω0t)\alpha(\omega_0 t)α(ω0​t) 要先乘進 Γ\GammaΓ， 變成 effective ISF，才是收音機真正的天線。

第 1 步：device noise 的兩種「頻段」​

一顆 MOS device 的汲極電流雜訊（折算到注入節點）大致分兩段（[P1] Eqs.(19),(22), p.185）：

Si(f)=in2‾Δf⏟white（平的）  +  in2‾Δf⋅ω1/f2πf⏟flicker（1/f）S_i(f)=\underbrace{\frac{\overline{i_n^2}}{\Delta f}}_{\text{white（平的）}}\;+\;\underbrace{\frac{\overline{i_n^2}}{\Delta f}\cdot\frac{\omega_{1/f}}{2\pi f}}_{\text{flicker（1/f）}}Si​(f)=white（平的）Δfin2​​​​​+flicker（1/f）Δfin2​​​⋅2πfω1/f​​​​
  • white noise：thermal（熱雜訊，4kTγgm4kT\gamma g_m4kTγgm​ 之類）+ shot，頻率平坦，能量散布在所有頻率， 包含 ω0,2ω0,…\omega_0,2\omega_0,\dotsω0​,2ω0​,… 附近。
  • flicker（1/f）noise：通道捕捉/釋放電荷的慢過程，能量集中在低頻（Δω≪ω0\Delta\omega\ll\omega_0Δω≪ω0​）， 在 device 1/f corner ω1/f\omega_{1/f}ω1/f​ 以下才顯著。
  • 單位：兩段都是 A²/Hz。

第 2 步：ISF 諧波是「接收頻道」——誰接誰​

把 ISF 傅立葉展開（[P1] Eq.(12)）代進 phase response（[P1] Eq.(13)），每個 cnc_ncn​ 項都是一個 把「nω0n\omega_0nω0​ 附近 noise」搬到 baseband 相位的 mixer：

  • white noise → 1/f²：所有 cnc_ncn​（n≥0n\ge0n≥0 的 white 部分）一起貢獻，透過 Parseval 收成 ∑n=0∞cn2=2Γrms2\sum_{n=0}^{\infty}c_n^2=2\Gamma_{rms}^2∑n=0∞​cn2​=2Γrms2​（[P1] Eq.(20), p.185），結果是 L∝Γrms2/qmax2\mathcal{L}\propto\Gamma_{rms}^2/q_{max}^2L∝Γrms2​/qmax2​（[P1] Eq.(21)）。所以 1/f² 由 Γrms\Gamma_{rms}Γrms​ 決定。

第 3 步：cyclostationary——device 不是一直在漏雜訊​

上面的 Γ\GammaΓ 是「裸 ISF」——它假設 noise 任何時刻都一樣大。但真實 device 的雜訊強度是 週期變化的（cyclostationary，週期穩態）：例如尾電流源只在某半週導通、switching pair 只在 transition 時 gmg_mgm​ 最大。Hajimiri–Lee 用一個 noise-modulating function（NMF，雜訊調變函數） α(ω0t)\alpha(\omega_0 t)α(ω0​t)（0≤α≤10\le\alpha\le10≤α≤1，週期）描述「device 何時在漏雜訊」，把它折進 ISF （[P1] Sec. II-D, Eq.(25)–(27), p.186；in(t)=in0(t)⋅α(ω0t)i_n(t)=i_{n0}(t)\cdot\alpha(\omega_0 t)in​(t)=in0​(t)⋅α(ω0​t) 為 Eq.(25)， Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α 為 Eq.(27)）：

 Γeff(ω0τ)=Γ(ω0τ)⋅α(ω0τ) \boxed{\ \Gamma_{eff}(\omega_0\tau)=\Gamma(\omega_0\tau)\cdot\alpha(\omega_0\tau)\ } Γeff​(ω0​τ)=Γ(ω0​τ)⋅α(ω0​τ) ​
  • 意義：真正進到 phase noise 公式的不是 Γ\GammaΓ，而是 Γeff\Gamma_{eff}Γeff​。所有 cnc_ncn​、c0c_0c0​、Γrms\Gamma_{rms}Γrms​ 都要用 Γeff\Gamma_{eff}Γeff​ 重算。
  • 設計上極關鍵的後果：就算波形對稱讓裸 Γ\GammaΓ 的 c0=0c_0=0c0​=0，如果 device 只在半週漏雜訊 （α\alphaα 不對稱），Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α 可能重新長出一個非零 c0c_0c0​ → flicker 又上轉了。 反之，把 device 的「導通窗口 α\alphaα」對準裸 Γ\GammaΓ 的低值相位，可以同時降 c0c_0c0​ 與 Γrms\Gamma_{rms}Γrms​。
  • 更嚴格的數學基礎（PPV / adjoint / Floquet）不在下載的 5 篇 PDF 內，以標準文獻補充 （Demir–Mehrotra–Roychowdhury 2000 等），見 effective_isf。

可用內建函式做 toy 計算（pedagogical toy model，非 transistor-level）：

import numpy as np
from simulations.common.isf_utils import (
    gamma_lc_ideal, effective_isf, compute_fourier_coefficients, gamma_rms)

theta = np.linspace(0, 2*np.pi, 2048, endpoint=False)
gamma = gamma_lc_ideal(theta)            # 裸 ISF = -sin(theta)，c0 = 0
# 不對稱 NMF：device 只在前半週導通（α=1 於 θ<π，否則 0）
alpha = np.where(theta < np.pi, 1.0, 0.0)
gamma_eff = effective_isf(gamma, alpha)       # 乘上 NMF（positional：gamma_values, alpha_values）
a0, a, b, c, ph = compute_fourier_coefficients(theta, gamma_eff, n_harmonics=8)
print("c0 =", a0, " Gamma_rms =", gamma_rms(theta, gamma_eff))
# -> c0 ≈ -0.637（裸 Γ 的 c0=0，被不對稱 α 重新長出來）、Gamma_rms ≈ 0.500

（完整 API 見 simulations/common/isf_utils.py；effective_isf 與 gamma_rms 為真實函式。）

第 4 步：兩個獨立槓桿——哪些 knob 改 Γrms\Gamma_{rms}Γrms​、哪些改 qmaxq_{max}qmax​​

快 transition（陡邊緣）→ 降 Γrms\Gamma_{rms}Γrms​提高 tank QQQ／RpR_pRp​（同電流更大 swing）

重點回顧​

  • device noise 分兩段：white（散布所有頻率）、flicker（集中低頻）。
  • ISF 諧波是收音頻道：c0c_0c0​ 收 flicker → 1/f³；c1,c2,…c_1,c_2,\dotsc1​,c2​,… 收 white → 1/f²（經 Parseval 收成 Γrms\Gamma_{rms}Γrms​）。
  • 真正起作用的是 effective ISF Γeff=Γ⋅α\Gamma_{eff}=\Gamma\cdot\alphaΓeff​=Γ⋅α（cyclostationary）；對稱的裸 Γ\GammaΓ 可能因不對稱 α\alphaα 又長出 c0c_0c0​。
  • 兩個獨立槓桿：改 Γrms\Gamma_{rms}Γrms​/c0c_0c0​（波形對稱、快邊緣、差動、α\alphaα 對齊）vs 改 qmaxq_{max}qmax​（swing、QQQ、differential）。
  • ring Γrms∝N−3/4\Gamma_{rms}\propto N^{-3/4}Γrms​∝N−3/4（[P2] Eq.(16), p.794，已核實）；固定 f0f_0f0​/功率下 phase noise ~與 NNN 無關。
