習題 4 解答（PLL 最佳 loop BW）

直覺。 PLL 把 ref 低通、VCO 高通（規範 10.2）。

  • loop BW fnf_nfn​ 太小 → VCO 的 close-in 1/f21/f^21/f2 沒被環路壓掉，in-band 全是 VCO 噪 → jitter 大。

    S_out, _, _ = shape_output_phase_noise(f, S_ref, S_vco, fn_hz=fn)
    var.append(np.trapezoid(S_out, f))       # rad^2

print("optimal f_n ~", f"{fn_opt:.2e}", "Hz")   # -> ~1.9e5 Hz（量級 10^5；非理想 brick-wall 故略低於 10/√S_ref≈3e5）

習題 7 解答（tail noise 對策，illustrative）

習題 8 解答（jitter 預算分配到 PLL 頻段）

設計反推策略。 RJ 各源不相關 → 方差（不是 rms）相加：

σt,tot2=σt,ref2+σt,vco2⟹σt,vco=σt,tot2−σt,ref2.\sigma_{t,\text{tot}}^2=\sigma_{t,\text{ref}}^2+\sigma_{t,\text{vco}}^2\quad\Longrightarrow\quad\sigma_{t,\text{vco}}=\sqrt{\sigma_{t,\text{tot}}^2-\sigma_{t,\text{ref}}^2}.σt,tot2​=σt,ref2​+σt,vco2​⟹σt,vco​=σt,tot2​−σt,ref2​​.

逐步代入（帶單位）。

σt,vco=(300 fs)2−(180 fs)2=90000−32400 fs=57600 fs=240 fs.\sigma_{t,\text{vco}}=\sqrt{(300\ \text{fs})^2-(180\ \text{fs})^2}=\sqrt{90000-32400}\ \text{fs}=\sqrt{57600}\ \text{fs}=240\ \text{fs}.σt,vco​=(300 fs)2−(180 fs)2​=90000−32400​ fs=57600​ fs=240 fs.

對應相位變異（用 σϕ=2πf0 σt\sigma_\phi=2\pi f_0\,\sigma_tσϕ​=2πf0​σt​，規範公式 19 反向）：

σϕ,vco=2πf0 σt,vco=2π×1010×240×10−15=1.508×10−2 rad,\sigma_{\phi,\text{vco}}=2\pi f_0\,\sigma_{t,\text{vco}}=2\pi\times10^{10}\times240\times10^{-15}=1.508\times10^{-2}\ \text{rad},σϕ,vco​=2πf0​σt,vco​=2π×1010×240×10−15=1.508×10−2 rad,σϕ,vco2=(1.508×10−2)2=2.274×10−4 rad2.\sigma_{\phi,\text{vco}}^2=(1.508\times10^{-2})^2=2.274\times10^{-4}\ \text{rad}^2.σϕ,vco2​=(1.508×10−2)2=2.274×10−4 rad2.

結果：VCO 預算 σt,vco=240\sigma_{t,\text{vco}}=240σt,vco​=240 fs；對應 σϕ,vco2≈2.27×10−4\sigma_{\phi,\text{vco}}^2\approx2.27\times10^{-4}σϕ,vco2​≈2.27×10−4 rad²。

手感：因為方差相加，180 fs + 240 fs（rms）合起來是 300 fs（不是 420 fs）——RJ 預算 要用平方和分配。這 240 fs 就是 PLL out-of-band VCO 段允許的積分 jitter，回頭定 loop BW 與 VCO 規格（連習題 4 的最佳 BW、pll_noise_budget）。

Dimension check：fs2−fs2=fs\sqrt{\text{fs}^2-\text{fs}^2}=\text{fs}fs2−fs2​=fs ✓；2πf0 σt2\pi f_0\,\sigma_t2πf0​σt​ 為 rad/s⋅s=rad\text{rad/s}\cdot\text{s}=\text{rad}rad/s⋅s=rad ✓。

import numpy as np
sigma_tot, sigma_ref, f0 = 300e-15, 180e-15, 10e9
sigma_vco = np.sqrt(sigma_tot**2 - sigma_ref**2)
sigma_phi = 2*np.pi*f0*sigma_vco
print(sigma_vco*1e15, "fs ;", sigma_phi**2, "rad^2")   # -> 240.0 fs ; 2.27e-4 rad^2
