Quadrature 產生與 coupled-oscillator phase noise

本頁定位（先講清楚）：這是一頁**進階（advanced）**設計頁。它把「怎麼生出一對相位差 90∘90^\circ90∘ 的時脈（quadrature，正交，即 I/Q）」這個 SerDes／收發機天天要面對的需求， 放回本站既有的 ISF + 廣義 Adler 機器裡解釋。前置（prerequisite）：先讀懂 [P1] 的 ISF（isf_definition、effective_isf）、 [P3] 的廣義 Adler 注入鎖定（paper_003）， 以及 [P4] 的 ILFD／頻率除法（paper_004）。 沒讀過這三頁，本頁的公式會看起來像憑空冒出來。

quadrature（正交，相位差 90∘90^\circ90∘ 的一對訊號 I 與 Q） 是現代收發機的基本元件：image-reject mixer（鏡像抑制混頻器）、單邊帶調變、half-rate（半速率）SerDes 的 4 相取樣、CDR 的相位偵測， 全都要一對乾淨、相位誤差小的 I/Q。問題是：產生 quadrature 本身要付 phase-noise 的代價，而且 不同產生法的代價結構完全不同。這頁要回答：

這頁要回答什麼：(1) 三種主流 quadrature 產生法各自的 phase-noise 代價長什麼樣？ (2) coupled QVCO（耦合正交壓控振盪器）裡，coupling strength（耦合強度）↔ I/Q phase error （正交相位誤差）↔ phase noise 為什麼是個三角權衡（trade-off）？(3) 耦合的那一對振盪器， 它們的雜訊是 common-mode（共模）還是 differential-mode（差模）相關，這如何決定那個有名的 ∼3\sim 3∼3 dB？(4) 怎麼把「耦合注入」這件事嚴格接回 [P3] 的 ISF／廣義 Adler？

物理直覺（先講結論）：耦合 QVCO 就是「兩個振盪器互相做 injection locking」——A 的輸出 注入 B、B 的輸出注入 A。一旦你這樣看，本站 [P3] 的整套機器立刻就能用：耦合電流 ici_{c}ic​ 看到的是 一個有效 ISF Γ~\tilde\GammaΓ~，它透過 Γ~\tilde\GammaΓ~ 的諧波 cnc_ncn​ 把對方的相位「拉」過來，鎖定動態 服從廣義 Adler 方程（[P3] Eq.(30)）。耦合越強，兩者相位越被綁死（phase error 越小），但耦合 device 本身也注入額外雜訊、又把頻率拉離 tank 共振點——這就是三角權衡的來源。

1. 三種 quadrature 產生法與它們的 phase-noise 代價​

先把三條路擺在一起。每條路的「代價」都不一樣，理解差異比記結論重要。

(a) Coupled QVCO（耦合正交 VCO）​

兩個相同的 LC VCO，用一對 coupling transistor（耦合電晶體） 把 A 的輸出注入 B、B 的輸出 （反相）注入 A，強迫兩者鎖在固定 90∘90^\circ90∘ 相位差。

  • 怎麼生 quadrature：兩個振盪器在「總迴路相位 =0=0=0」的約束下，被耦合方式強迫成 90∘90^\circ90∘（推導見第 4 節，這是 [P3] 廣義 Adler 在「互注入」情形的穩態解）。
  • phase-noise 代價：理論上可以比單顆好——兩顆相同 VCO 的差動／反相耦合會讓不相關的 雜訊部分平均掉（∼3\sim 3∼3 dB 的潛在好處，第 3 節）；但耦合 device 自身的雜訊會打進來，且 強耦合把頻率拉離 tank peak、降低有效 QQQ，反而抬高 phase noise。net 是好是壞，取決於耦合強度 （第 2 節的三角權衡）。

(b) Divide-by-2（÷2）from a 2f02f_02f0​ source（ILFD／靜態除頻器）​

跑一顆 2f02f_02f0​ 的振盪器，再除以 2。一個 master-slave 觸發器除頻器天生輸出兩個相位差 90∘90^\circ90∘ 的時脈（因為 ÷2\div 2÷2 把一個輸入週期映成輸出半週期 =90∘=90^\circ=90∘）；或用 ILFD（injection-locked frequency divider，注入鎖定除頻器）——把 2f02f_02f0​ 注入一顆 f0f_0f0​ 振盪器，靠 ISF 的第 2 諧波鎖定 到 f0f_0f0​（這正是 paper_004 講的 ILFD，subharmonic locking）。

  • 怎麼生 quadrature：÷2\div 2÷2 的兩個互補輸出（或差動 ÷2 的四個節點）天然相差 90∘90^\circ90∘， 正交精度由電路對稱性決定，與 tank 失諧無關——這是它相對 QVCO 的最大優點。

(c) RC-CR polyphase filter（多相濾波器）​

純被動：一個 RC 低通給一路（相位 −45∘-45^\circ−45∘）、一個 CR 高通給另一路（相位 +45∘+45^\circ+45∘），在 ω=1/RC\omega=1/RCω=1/RC 處兩路差 90∘90^\circ90∘。多級串接（polyphase）可拓寬頻寬。

  • 怎麼生 quadrature：被動相移，90∘90^\circ90∘ 由 RRR、CCC 匹配決定；不需要第二顆振盪器。
  • phase-noise 代價：被動網路不產生新的 phase noise（理論上），但它有兩個代價：(1) 插入損耗（insertion loss）——每級 RC-CR 衰減 ∼3\sim 3∼3 dB，要補一級 buffer，buffer 的 thermal noise 變成 additive noise（加成雜訊，劣化 SNR、抬高遠端 noise floor）；(2) 正交精度對 RRR、CCC 的絕對值與頻率敏感——偏離 1/RC1/RC1/RC 就有 I/Q 相位與振幅誤差，要靠多級 與校正。它不改 close-in（近載波）的 1/f21/f^21/f2、1/f31/f^31/f3 phase noise（那是 PLL／VCO 決定的）， 只在遠端因 buffer 加成雜訊抬一點 floor。

2. Coupled QVCO：coupling strength ↔ I/Q phase error ↔ phase noise​

這是 QVCO 設計的核心三角權衡。先定義耦合強度，再說明三者怎麼互相拉扯。

耦合強度 mmm 的定義​

設每顆 VCO 的核心（−GmG_mGm​）跨導讓它自己起振、提供電流 IcoreI_{core}Icore​；耦合電晶體把對方的訊號 注入本顆，注入電流幅度 IcI_{c}Ic​。定義耦合係數（coupling factor）

m≡IcIcorem\equiv\frac{I_{c}}{I_{core}}m≡Icore​Ic​​

mmm 是無因次的「耦合相對核心有多強」。m→0m\to 0m→0 兩顆解耦、各跑各的；mmm 大則強綁。

三者的拉扯（physics）​

(i) coupling ↑ → I/Q phase error ↓：耦合提供把兩顆「拉回 90∘90^\circ90∘」的恢復力。任何讓兩顆 本徵頻率不同的失配（mismatch，LLL、CCC、gmg_mgm​ 的製程偏差）都會試圖把相位差推離 90∘90^\circ90∘； 耦合越強，恢復力越大，殘留相位誤差越小。直覺與量級（外部文獻，QVCO 標準結果）：

(ii) coupling ↑ → phase noise ↑（兩個機制）：

  • 耦合 device 自身注入雜訊：耦合電晶體和核心 device 一樣有 thermal／flicker noise，它直接打在 tank 節點上、看到一個有效 ISF，貢獻額外的 phase noise（第 4 節用 ISF 量化）。耦合越強（耦合 device 越大、電流越大），這份雜訊越大。
  • 耦合把振盪頻率拉離 tank peak、降低有效 QQQ：耦合注入是一股與 tank 電壓不同相位的電流， 振盪器必須偏離 tank 共振頻率才能讓總迴路相位 =0=0=0（這是 injection pulling 的穩態）。偏離共振 → tank 在工作頻率的相位斜率（即有效 QQQ）變小 → 由 Leeson／ISF，phase noise ∝1/Q2\propto 1/Q^2∝1/Q2 抬高。這是強耦合 QVCO 的主要 phase-noise 懲罰。

(iii) net trade-off：所以 QVCO 設計是在「夠強的耦合壓低 I/Q phase error」與「過強的耦合 抬高 phase noise」之間找甜蜜點。實務經驗（外部文獻）：存在一個最佳 mmm（典型 m∼0.2 ⁣− ⁣0.5m\sim 0.2\!-\!0.5m∼0.2−0.5 量級，視拓樸），太小則 I/Q error 與失鎖風險大，太大則 phase noise 被有效 QQQ 降低主導。

設計旋鈕（coupled QVCO）：

  1. 耦合強度 mmm：折衷 I/Q error（要 mmm 大）與 phase noise（要 mmm 小）→ 取中間最佳值。
  2. 耦合接法：series 一般優於 parallel（頻率拉移小、不另耗流）。
  3. 降耦合 device 雜訊：耦合電晶體用大尺寸、低 gmg_mgm​／低 flicker，減少它注入的 Γrms\Gamma_{rms}Γrms​。
  4. 降失配：好的 layout、共質心（common-centroid）→ 同樣 mmm 下 I/Q error 更小，可用較弱耦合。

3. 雜訊相關性：common-mode vs differential-mode 與 ~3 dB​

QVCO 是「兩顆相同振盪器」，所以必須分清楚哪些雜訊在兩顆之間相關、哪些不相關——這決定那個 有名的 ∼3\sim 3∼3 dB 到底是賺還是賠。

兩種雜訊的記帳​

把第二顆（Q）想成第一顆（I）的「複製品」。對 I/Q 兩路各自的 phase noise 功率：

  • differential-mode / 不相關雜訊（每顆 VCO 核心 device 自己的 thermal noise，兩顆獨立）： 兩顆的相位雜訊不相關。當你把兩顆強耦合綁成同一個振盪系統，輸出的相位由兩顆共同決定 ——兩份不相關功率平均，輸出 phase noise 比單顆低 ∼3\sim 3∼3 dB（10log⁡10210\log_{10}210log10​2）。這是 QVCO「用兩顆換 3 dB」的理論好處。
LQVCO ≈ Lsingle−10log⁡102 = Lsingle−3.01 dB（僅就不相關核心雜訊而言）\mathcal{L}_{QVCO}\ \approx\ \mathcal{L}_{single}-10\log_{10}2\ =\ \mathcal{L}_{single}-3.01\ \text{dB}\quad\text{（僅就不相關核心雜訊而言）}LQVCO​ ≈ Lsingle​−10log10​2 = Lsingle​−3.01 dB（僅就不相關核心雜訊而言）
  • common-mode / 相關雜訊（同時打在兩顆、或耦合路徑共用的雜訊，例如共用 tail、共用 bias、 耦合 device）：兩顆的擾動完全相關，平均不會降低它——相關功率不享受 N\sqrt{N}N​ 平均。 這份雜訊抵銷掉部分 3 dB 好處。

帳要怎麼算（關鍵 caveat）​

∼3\sim 3∼3 dB 的誠實版本：「兩顆 → −3-3−3 dB」只對不相關雜訊成立，且前提是耦合 device 不另 引進可觀的相關雜訊、也不把有效 QQQ 拉低。實務上：

  1. 核心 device 的不相關 thermal noise：享受 −3-3−3 dB ✓。
  2. 耦合 device 注入的雜訊：是新增的、且常相關（共用耦合路徑）→ 抵銷部分好處。
  3. 有效 QQQ 因耦合拉頻而降：phase noise ∝1/Q2\propto 1/Q^2∝1/Q2 抬高 → 再吃掉一些。

所以真實 QVCO 不一定比單顆好 3 dB，可能只好 1–2 dB，甚至在強耦合下更差。∼3\sim 3∼3 dB 是 「兩顆不相關源平均」的上限，不是保證值（外部文獻，非本站 5 篇 PDF）。

記帳口訣：相關 ↔ 振幅相加（總功率 ∝N2\propto N^2∝N2，相對單源不下降）→ 不享受平均； 不相關 ↔ 功率相加（總功率 ∝N\propto N∝N）→ 享受 −10log⁡10N-10\log_{10}N−10log10​N。 這跟第 1 節 ÷N\div N÷N 改善 20log⁡10N20\log_{10}N20log10​N（相位被除以 NNN，是 deterministic 的相關縮放）是同一套相關性記帳邏輯的兩面。

4. 把耦合注入接回 ISF / 廣義 Adler（[P3]）​

這是本頁的數學核心：耦合 QVCO 的鎖定動態，就是 [P3] 的廣義 Adler 方程套在「互注入」上。 我們不另造新理論，直接把本站 paper_003 的式子拿來用。

第 1 步：耦合電流看到的有效 ISF​

從 A 注入 B 的耦合電流 ic(t)i_{c}(t)ic​(t)，和任何注入電流一樣，透過有單位 ISF Γ~(x)=Γ(x)/qmax\tilde\Gamma(x)=\Gamma(x)/q_{max}Γ~(x)=Γ(x)/qmax​（單位 rad/C，[P3] Eq.(26), p.2113）推動 B 的相位。也就是說， 耦合注入不是什麼特別的東西——它看到的就是 B 這顆振盪器在注入節點的有效 Γ~\tilde\GammaΓ~，並透過 Γ~\tilde\GammaΓ~ 的傅立葉諧波 cnc_ncn​（[P1] Eq.(12)）把對方的訊號「收」進來：諧波對齊得越好，耦合 （鎖定）越有效。

耦合 device 自身的雜訊 ic,n(t)i_{c,n}(t)ic,n​(t) 也走同一條 Γ~\tilde\GammaΓ~，所以它對 phase noise 的貢獻就是 [P1] 那套：L∝Γrms2/qmax2\mathcal{L}\propto\Gamma_{rms}^2/q_{max}^2L∝Γrms2​/qmax2​（[P1] Eq.(21), p.185，把 Γ\GammaΓ 換成耦合注入 看到的有效 ISF）。這就是第 2 節「耦合 device 注入額外雜訊」的量化出口。

第 2 步：互注入的廣義 Adler​

把 B 對 A 的注入也寫上。對振盪器 A，相對相位 θA=ϕA−ϕref\theta_A=\phi_A-\phi_{ref}θA​=ϕA​−ϕref​ 服從 [P3] 的時間平均 廣義 Adler 方程（[P3] Eq.(30), p.2113，平均項前為加號）：

dθAdt=(ω0,A−ωref)+1T∫TΓ~(ωreft+θA) ic,B→A(t) dt\frac{d\theta_A}{dt}=(\omega_{0,A}-\omega_{ref})+\frac{1}{T}\int_{T}\tilde\Gamma\big(\omega_{ref}t+\theta_A\big)\,i_{c,B\to A}(t)\,dtdtdθA​​=(ω0,A​−ωref​)+T1​∫T​Γ~(ωref​t+θA​)ic,B→A​(t)dt

其中 ic,B→Ai_{c,B\to A}ic,B→A​ 是 B 注入 A 的耦合電流（正比於 B 的輸出 ∝cos⁡(ωreft+θB)\propto\cos(\omega_{ref}t+\theta_B)∝cos(ωref​t+θB​)）。 對 B 對稱地寫一條。重點：耦合 QVCO 不過是兩條這樣的 Adler 方程互相耦合——A 的方程裡 出現 θB\theta_BθB​、B 的方程裡出現 θA\theta_AθA​。整理成 [P3] 的 lock characteristic 形式 （[P3] Eq.(33), p.2114）：

dθAdt=(ω0,A−ωref)+Ω(θA−θB)\frac{d\theta_A}{dt}=(\omega_{0,A}-\omega_{ref})+\Omega(\theta_A-\theta_B)dtdθA​​=(ω0,A​−ωref​)+Ω(θA​−θB​)

Ω(⋅)\Omega(\cdot)Ω(⋅) 即耦合造成的平均頻率偏移隨相對相位差的函數（耦合越強，Ω\OmegaΩ 的值域越寬， 鎖定越牢）。

第 3 步：穩態解 → 為什麼是 90∘90^\circ90∘​

QVCO 的耦合方式（A 同相注入 B、B 反相注入 A，差一個負號）使得穩態 （dθA/dt=dθB/dt=0d\theta_A/dt=d\theta_B/dt=0dθA​/dt=dθB​/dt=0、兩顆同頻）下，兩條 Adler 方程只能在相位差

θA−θB=±90∘\theta_A-\theta_B=\pm 90^\circθA​−θB​=±90∘

時自洽——正號或負號（I 領先或落後 Q）是兩個對稱的穩定解之一，由起振瞬間決定。90∘90^\circ90∘ 不是 湊出來的，是互注入 Adler 方程的穩態約束。任何 tank 失諧（ω0,A≠ω0,B\omega_{0,A}\ne\omega_{0,B}ω0,A​=ω0,B​）會把這個 90∘90^\circ90∘ 推偏一點，偏多少由 Ω\OmegaΩ 的斜率（即耦合強度）抵抗——這正是第 2 節 ΔϕIQ∝Q/m\Delta\phi_{IQ}\propto Q/mΔϕIQ​∝Q/m 的微分方程版本。

量級／dimension check：Ω(θ)\Omega(\theta)Ω(θ) 與 (ω0−ωref)(\omega_0-\omega_{ref})(ω0​−ωref​) 同單位 rad/s ✓。 耦合強 → Ω\OmegaΩ 值域寬 → 同樣失諧 (ω0,A−ω0,B)(\omega_{0,A}-\omega_{0,B})(ω0,A​−ω0,B​) 只需很小的相位偏移就能 由 Ω(θA−θB)\Omega(\theta_A-\theta_B)Ω(θA​−θB​) 補回 → I/Q 誤差小。耦合弱 → Ω\OmegaΩ 窄 → 失諧一大就無解（失鎖， 兩顆不再同頻、相位週期滑動，這就是 [P3] 講的 injection pulling）。

重點回顧​

  • 三種 quadrature 產生法的代價結構不同：coupled QVCO 的代價是耦合 device 雜訊 + 拉頻降 QQQ（淨值看耦合強度）； ÷2\div 2÷2（ILFD）理想改善 −20log⁡10N-20\log_{10}N−20log10​N，但要先有乾淨的 2f02f_02f0​ 源；RC-CR polyphase 不產生新 close-in PN， 代價是插損 + buffer additive noise + 頻寬窄。
  • coupled QVCO 的三角權衡：coupling ↑ → I/Q phase error ↓（恢復力強），但 phase noise ↑（耦合 device 注雜訊 + 拉頻降有效 QQQ）→ 存在最佳耦合強度 mmm。series coupling 一般優於 parallel（頻率拉移小、不另耗流）。
  • ∼3\sim 3∼3 dB 是上限不是保證：只有不相關的核心 device 雜訊享受 −10log⁡102-10\log_{10}2−10log10​2；耦合/共用雜訊是相關的， 抵銷部分好處；拉頻降 QQQ 再吃掉一些 → 真實 QVCO 常只好 1–2 dB，強耦合下甚至更差。
  • 接回 [P3]：耦合 QVCO ＝ 兩條互注入的廣義 Adler 方程（[P3] Eq.(30), p.2113）；耦合電流走 有單位 ISF Γ~=Γ/qmax\tilde\Gamma=\Gamma/q_{max}Γ~=Γ/qmax​（[P3] Eq.(26)），90∘90^\circ90∘ 是反相互注入的穩態約束， lock range ＝ Ω(θ)\Omega(\theta)Ω(θ) 值域（[P3] Eq.(33)）。鎖定與額外 phase noise 是同一個 Γ~\tilde\GammaΓ~ 的兩面。
  • 誠實標註：QVCO 的拓樸比較、ΔϕIQ\Delta\phi_{IQ}ΔϕIQ​ 量級式、∼3\sim 3∼3 dB、parallel/series 結論皆為 外部文獻（非本站 5 篇 PDF）；其鎖定動態嚴格繫於 [P3]，÷2\div 2÷2/ILFD 的 subharmonic 機制嚴格繫於 [P4]。
